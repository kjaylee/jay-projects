## 💡 HTML5 게임에서 사용하기

### Base64 인코딩
```javascript
// 이미지를 Base64로 변환하여 HTML에 직접 삽입
const img = new Image();
img.src = 'data:image/png;base64,iVBORw0KGgo...';
```

### 크레딧 표기 예시
```html
<!--
Asset Credits:
- Sprites: Kenney.nl "Pixel Platformer" (CC0)
- UI: Kenney.nl "UI Pack" (CC0)
- Audio: Kenney.nl "Interface Sounds" (CC0)
- Font: Google Fonts "Press Start 2P"
-->
```
