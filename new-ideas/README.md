# 🤖 새 아이디어

> 새로운 프로젝트 아이디어를 기록하는 공간입니다.

---

## 📝 아이디어 추가 방법

1. 이 폴더에 마크다운 파일 생성 (예: `01_아이디어명.md`)
2. `_sidebar.md` 파일에 링크 추가
3. 웹사이트 새로고침

---
