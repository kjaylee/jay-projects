# 🟢 슬라임 서바이버 (Slime Survivor)

## 📋 기본 정보
- **장르**: 럭 배틀러 + 로그라이크
- **플랫폼**: iOS / Android
- **타겟**: 캐주얼~미드코어 (15-35세)
- **예상 개발기간**: 4-6개월
- **팀 규모**: 2-4명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "귀여운 슬라임을 수집하고 합성해서 던전을 자동으로 클리어하는 오토배틀 로그라이크"

### 핵심 재미 요소
1. **수집의 즐거움** - 100종 이상의 슬라임 도감
2. **합성 도박** - 랜덤 결과의 슬라임 합성
3. **전략적 배치** - 슬라임 포지션이 승패 결정
4. **런 기반 성장** - 매 런마다 새로운 빌드

---

## ⚙️ 핵심 메카닉

### 1. 슬라임 시스템
```
[등급]
- 커먼 (50%) → 언커먼 (30%) → 레어 (15%) → 에픽 (4%) → 레전더리 (1%)

[속성]
- 불 > 풀 > 물 > 불 (상성 시스템)
- 빛 ↔ 어둠 (상호 강함)

[슬라임 예시]
- 🔴 파이어젤리: 기본 화염 공격
- 🔵 아쿠아블롭: 힐 + 버프
- 🟢 리프슬라임: 독 + 지속 데미지
- ⚡ 썬더푸딩: 연쇄 공격
```

### 2. 합성 시스템
```
같은 등급 슬라임 3개 → 상위 등급 1개 (랜덤 결과)

[특수 레시피]
파이어젤리 + 아쿠아블롭 + 리프슬라임 = 레인보우슬라임 (레전더리)
```

### 3. 전투 시스템
- **자동 전투** - 배치만 하면 자동 진행
- **3x3 그리드** - 9칸에 슬라임 배치
- **포지션 효과** - 앞줄(공격↑), 뒷줄(방어↑), 중앙(스킬↑)

### 4. 런 구조
```
[1런 = 10스테이지]
- 스테이지 1-3: 이지 (슬라임 드랍)
- 스테이지 4-6: 노멀 (장비 드랍)
- 스테이지 7-9: 하드 (재화 드랍)
- 스테이지 10: 보스 (보장 레어+)

[런 사이]
- 영구 업그레이드 구매
- 슬라임 합성
- 덱 구성
```
