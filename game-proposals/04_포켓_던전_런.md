# 🗡️ 포켓 던전 런 (Pocket Dungeon Run)

## 📋 기본 정보
- **장르**: 로그라이크 러너 + 덱빌딩
- **플랫폼**: iOS / Android
- **타겟**: 코어 캐주얼 (20-40세)
- **예상 개발기간**: 5-6개월
- **팀 규모**: 2-4명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "달리면서 카드를 모아 덱을 빌딩! 매 런마다 다른 빌드로 던전을 돌파하는 로그라이크 러너"

### 핵심 재미 요소
1. **런 중 덱빌딩** - 달리면서 카드 수집
2. **자동 전투** - 수집한 카드로 오토 배틀
3. **시너지 빌드** - 카드 조합으로 폭딜
4. **영구 성장** - 언락 콘텐츠

---

## ⚙️ 핵심 메카닉

### 1. 러닝 페이즈
```
[조작]
- 좌/우 스와이프: 레인 이동
- 탭: 점프
- 3레인 러너 구조

[러닝 중 수집]
💎 골드: 상점에서 카드 구매
🃏 카드: 즉시 덱에 추가
💊 포션: 즉시 회복
⭐ 경험치: 런 종료 후 영구 성장
```

### 2. 카드 시스템
```
[카드 타입]
⚔️ 공격: 데미지 딜링
🛡️ 방어: 쉴드/회피
🔮 마법: 특수 효과
💚 회복: 체력 복구
⚡ 버프: 스탯 강화

[등급]
커먼(흰) → 언커먼(초록) → 레어(파랑) → 에픽(보라) → 레전더리(금)

[카드 예시]
- 베기 (커먼): 10 데미지
- 연속베기 (레어): 8 데미지 x 3회
- 처형 (에픽): 적 체력 20% 이하 시 즉사
- 오버드라이브 (레전더리): 이번 턴 모든 데미지 3배
```

### 3. 전투 페이즈
```
[트리거]
러닝 중 적 조우 → 자동 전투 시작

[전투 흐름]
1. 덱에서 카드 5장 드로우
2. 왼쪽부터 순서대로 자동 사용
3. 적 턴 → 내 턴 반복
4. 승리 시 보상 + 러닝 재개

[시너지]
같은 타입 카드 3장 → 보너스 효과
예: 공격 카드 3장 → 전체 공격력 +30%
```

### 4. 상점 시스템
```
[런 중 상점]
일정 거리마다 상점 등장
- 카드 구매 (골드)
- 카드 제거 (골드)
- 카드 업그레이드 (골드)
- 체력 회복 (골드)

[카드 업그레이드]
베기 → 베기+ (10 → 15 데미지)
```

### 5. 던전 구조
```
[1런 = 3존]
존 1: 숲 (3 스테이지 + 미니보스)
존 2: 동굴 (3 스테이지 + 미니보스)
존 3: 성 (3 스테이지 + 최종보스)

[스테이지 = 러닝 + 전투 반복]
약 500m 러닝 + 전투 2-3회
```
