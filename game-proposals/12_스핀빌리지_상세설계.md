## 1. 슬롯 시스템 상세

### 1.1 릴 구성

```
[슬롯 구조]
┌─────┬─────┬─────┐
│ 릴1 │ 릴2 │ 릴3 │
└─────┴─────┴─────┘

[각 릴의 심볼 배치 (20칸)]
릴 1: 🪙🪙🪙🪙🪙🪙⚔️⚔️⚔️🛡️🛡️💰💰💰👑⭐⭐🪙🪙⚔️
릴 2: 🪙🪙🪙🪙🪙⚔️⚔️⚔️🛡️🛡️🛡️💰💰👑👑⭐🪙🪙⚔️💰
릴 3: 🪙🪙🪙🪙🪙🪙⚔️⚔️🛡️🛡️💰💰💰👑⭐⭐⭐🪙🪙⚔️
```

| 조합 | 보상 | 설명 |

```
[스핀 시퀀스]
1. 버튼 탭 → 버튼 축소 피드백 (0.1초)
```

## 2. 건물 업그레이드 시스템

### 2.1 건물 구조

```
[마을당 건물 슬롯: 5개]
┌─────────────────────────────┐
│        [메인 건물]          │
│           🏰               │
├─────────────────────────────┤
│  [건물1] [건물2] [건물3]    │
│    🏠     🏪      🏛️       │
├─────────────────────────────┤
│       [특수 건물]           │
│          ⛪                │
└─────────────────────────────┘
```

### 2.2 업그레이드 비용 공식

```
업그레이드 비용 = 기본비용 × 마을계수 × 레벨계수

기본비용: 100만 코인

마을계수:
- 마을 1-10:   1.0 ~ 2.5
- 마을 11-30:  3.0 ~ 10.0
- 마을 31-50:  12.0 ~ 50.0
- 마을 51-100: 60.0 ~ 500.0

레벨계수:
- 레벨 1: 1.0
- 레벨 2: 2.5
- 레벨 3: 6.0
- 레벨 4: 15.0
- 레벨 5: 40.0
```

### 2.3 마을 1 예시 비용표

| 건물 | 레벨1 | 레벨2 | 레벨3 | 레벨4 | 레벨5 | 합계 |
|------|-------|-------|-------|-------|-------|------|
| 농가 | 100만 | 250만 | 600만 | 1500만 | 4000만 | 6450만 |
| 헛간 | 120만 | 300만 | 720만 | 1800만 | 4800만 | 7740만 |
| 우물 | 80만 | 200만 | 480만 | 1200만 | 3200만 | 5160만 |
| 풍차 | 150만 | 375만 | 900만 | 2250만 | 6000만 | 9675만 |
| 마을회관 | 200만 | 500만 | 1200만 | 3000만 | 8000만 | 12900만 |
| **총합** | - | - | - | - | - | **4억 1925만** |

### 2.4 건물 업그레이드 연출

```
[업그레이드 시퀀스]
1. 건물 탭 → 업그레이드 UI 오픈
2. 코인 지불 확인 → 코인 날아가는 연출
3. 건설 이펙트 (먼지, 망치질 사운드)
4. 건물 모델 교체 (레벨업된 버전)
5. 별 이펙트 + 레벨업 텍스트
6. 진행률 바 업데이트

소요 시간: 2-3초
```

### 2.5 건물 파괴/수리 시스템

```
[파괴 시]
- 공격받으면 건물 레벨 -1
- 레벨 0이 되면 건물 파손 상태 (회색 표시)
- 파손 상태에서 추가 공격 시 레벨 유지 (0 이하 안됨)

[수리 비용]
- 해당 레벨 업그레이드 비용의 50%
- 예: 레벨 3→4 업그레이드가 1500만이면, 수리비 750만
```

## 3. 마을 레벨업 시스템

### 3.1 마을 완성 조건

```
[완성 조건]
- 모든 건물 5개가 레벨 5 달성

[완성 시 보상]
- 다음 마을 해금
- 카드 팩 1개 (세트 관련)
- 보너스 코인 (마을 번호 × 100만)
- 스핀 20개
- 경험치 (플레이어 레벨업용)
```

### 3.2 마을 테마 상세

| 마을 번호 | 테마 | 건물 구성 | 시각적 특징 |
|-----------|------|----------|-------------|
| 1-5 | 농촌 | 농가,헛간,우물,풍차,마을회관 | 초록 풀밭, 밝은 톤 |
| 6-10 | 어촌 | 선착장,등대,어시장,배,항구청 | 푸른 바다, 해변 |
| 11-20 | 중세 | 성,기사단,대장간,마법탑,주점 | 돌벽, 깃발 |
| 21-30 | 해적 | 해적선,보물창고,선술집,감옥,망루 | 열대 섬, 야자수 |
| 31-40 | 동화 | 과자집,버섯집,요정탑,마법숲,왕궁 | 판타지 색감 |
| 41-50 | 동양 | 도장,탑,정원,시장,왕궁 | 한옥/중국풍 |
| 51-70 | 사막 | 피라미드,오아시스,시장,궁전,스핑크스 | 황금빛, 모래 |
| 71-100 | 미래 | 우주선,연구소,로봇공장,에너지탑,스테이션 | 네온, SF |
| 100+ | 신화 | 올림푸스,발할라,천궁 등 | 신비로운 분위기 |

### 3.3 마을 이동 애니메이션

```
[마을 완성 → 이동 시퀀스]
1. "축하합니다!" 배너 + 불꽃놀이
2. 완성된 마을 360도 회전 뷰
3. 보상 팝업 (카드, 코인, 스핀)
4. 화면 페이드아웃
5. 새 마을 줌인 연출
6. 건물 슬롯 하나씩 등장
7. "새로운 모험을 시작하세요!" 메시지

총 소요: 8-10초 (스킵 가능)
```

## 4. 공격/방어/레이드 시스템

### 4.1 공격 시스템 상세

```
[공격 대상 선택]
1. 랜덤 대상: 자동 매칭된 유저
2. 친구 대상: 친구 목록에서 선택
3. 복수 대상: 나를 공격한 유저 (복수 버프 적용)

[공격 프로세스]
1. 해머 3개 일치 → 공격 권한 획득
2. 대상 마을로 이동 (줌아웃 → 이동 → 줌인)
3. 건물 5개 중 1개 자동 선택 (랜덤)
4. 해머 날아가는 연출
5. 방패 체크
   - 방패 있음 → 방패 깨지는 연출, 건물 무사
   - 방패 없음 → 건물 타격, 레벨 -1
6. 보상 코인 획득 (건물 레벨 × 10만)
```

### 4.2 방패 시스템 상세

```
[방패 저장]
- 최대 보유: 3개
- 획득: 슬롯 방패 심볼, 이벤트, 구매

[방패 작동]
- 공격 시 자동 소모 (1공격 = 1방패)
- 방패 없는 상태에서 공격받으면 건물 피해

[방패 알림]
- 방패 0개 시 경고 아이콘
- 푸시: "방패가 없어요! 빨리 스핀하세요!"
```

### 4.3 레이드 시스템 상세

```
[레이드 프로세스]
1. 돼지 3개 일치 → 레이드 권한 획득
2. 대상 마을로 이동
3. 코인 숨김 위치 3개 표시 (X표시 땅)
4. 플레이어가 1개 선택 (탭)
5. 돼지가 땅 파는 연출
6. 코인 발견 → 보상 획득

[레이드 보상 계산]
기본 레이드 = 대상 보유 코인의 10%
메가 레이드 (💰💰💰) = 대상 보유 코인의 30%

최소 보상: 100만 코인
최대 보상: 5억 코인

[펫 보너스]
- 럭키 피기: +20%
- 골드 드래곤: +50%
```

### 4.4 복수 시스템

```
[복수 트리거]
- 누군가 나를 공격/레이드 시 복수 버튼 활성화
- 복수 목록 최대 10명 저장

[복수 보너스]
- 공격 데미지 x1.5
- 레이드 보상 x1.5
- 복수 성공 시 "복수 완료!" 뱃지

[복수 알림]
- 푸시: "[닉네임]이 당신 마을을 공격했어요! 복수하세요!"
- 앱 내 레드닷 표시
```

## 5. 카드 컬렉션 시스템

### 5.1 카드 구조

```
[세트 구성]
- 총 세트 수: 50개 (확장 가능)
- 세트당 카드: 9장
- 총 카드: 450장

[카드 등급 분포 (세트당)]
- ⬜ 커먼: 4장 (드롭률 60%)
- 🟦 레어: 3장 (드롭률 28%)
- 🟪 에픽: 1장 (드롭률 10%)
- 🟨 레전더리: 1장 (드롭률 2%)
```

### 5.2 카드 획득 방법

| 방법 | 획득량 | 비고 |
|------|--------|------|
| 마을 완성 | 카드팩 1개 (카드 4장) | 마을 관련 세트 |
| 일일 보상 | 카드 1장 (랜덤) | 매일 로그인 |
| 이벤트 | 특수 카드팩 | 이벤트 한정 |
| 친구 교환 | 1:1 교환 | 중복 카드만 |
| 구매 | 카드팩 구매 | IAP |

### 5.3 세트 완성 보상

| 세트 등급 | 완성 보상 |
|-----------|----------|
| 커먼 세트 (1-10번) | 스핀 100개 + 코인 1억 |
| 레어 세트 (11-25번) | 스핀 200개 + 코인 5억 + 펫 에그 |
| 에픽 세트 (26-40번) | 스핀 400개 + 코인 20억 + 희귀 펫 |
| 레전더리 세트 (41-50번) | 스핀 1000개 + 코인 100억 + 전설 펫 |

### 5.4 카드 교환 시스템

```
[교환 규칙]
- 같은 등급만 교환 가능
- 중복 카드만 교환에 사용 가능
- 하루 교환 횟수: 5회 (친구당 1회)

[교환 UI]
1. 친구 프로필 → "카드 교환" 탭
2. 내 중복 카드 선택
3. 원하는 카드 요청
4. 친구가 수락하면 교환 완료

[부족 카드 알림]
- "이 카드를 가진 친구: 3명"
- 바로 요청 버튼
```

## 6. 펫 시스템

### 6.1 펫 종류 및 스킬

| 펫 | 등급 | 패시브 스킬 | 해금 조건 |
|----|------|------------|----------|
| 🐷 럭키 피기 | 커먼 | 레이드 보상 +20% | 기본 지급 |
| 🐶 코인 하운드 | 커먼 | 코인 심볼 보상 +15% | 마을 5 완성 |
| 🐱 쉴드 캣 | 레어 | 방패 심볼 확률 +10% | 마을 15 완성 |
| 🐯 스트라이크 타이거 | 레어 | 공격 시 추가 공격 15% | 마을 25 완성 |
| 🦊 트릭시 폭스 | 에픽 | 레이드 시 보너스 위치 노출 | 마을 40 완성 |
| 🐉 골드 드래곤 | 에픽 | 모든 코인 보상 +50% | 카드 세트 10개 완성 |
| 🦄 매직 유니콘 | 레전더리 | 방패 무시 공격 20% | 이벤트 한정 |
| 🐺 섀도우 울프 | 레전더리 | 복수 데미지 x3 | 마을 100 완성 |
| 🦅 스카이 피닉스 | 레전더리 | 마을 완성 보상 x2 | 시즌패스 100레벨 |

### 6.2 펫 레벨업 시스템

```
[레벨업 방식]
- 중복 펫 합성으로 레벨업
- 같은 펫 2마리 = 레벨 +1

[레벨당 스킬 증가]
레벨 1: 기본 스킬
레벨 2: 스킬 효과 +25%
레벨 3: 스킬 효과 +50%
레벨 4: 스킬 효과 +75%
레벨 5 (MAX): 스킬 효과 +100% + 특수 외형

[예시: 럭키 피기]
Lv1: 레이드 보상 +20%
Lv2: 레이드 보상 +25%
Lv3: 레이드 보상 +30%
Lv4: 레이드 보상 +35%
```

## 9. 밸런싱 수치

### 9.1 경제 밸런싱

```
[스핀 경제]
- 시간당 자연 충전: 6스핀
- 일일 무료 스핀: 약 144스핀 (자연) + 50스핀 (친구) = 194스핀
- 일일 권장 플레이: 300스핀 (부족분 광고/구매로 충당)

[코인 순환]
마을 1 완성 필요 코인: 4.2억
일일 평균 코인 획득: 약 1억 (300스핀 기준)
마을 1 완성 예상 일수: 4-5일

[마을별 완성 소요 일수]
마을 1-10:  3-5일/마을
마을 11-30: 5-7일/마을
마을 31-50: 7-10일/마을
마을 51+:   10-14일/마을
```

### 9.2 슬롯 RTP (Return To Player)

```
[RTP 계산]
이론적 RTP: 85%
- 100스핀 평균 → 85스핀 가치의 보상 획득
- 나머지 15%는 시스템 흡수 (싱크)

[싱크 보정]
- 초반 마을: RTP 95% (신규 유저 유지)
- 중반 마을: RTP 85% (표준)
- 후반 마을: RTP 80% (과금 유도)
```

### 9.3 잭팟 확률 밸런싱

```
[잭팟 타이밍]
- 평균 800스핀당 1회 소형 잭팟
- 평균 5000스핀당 1회 대형 잭팟
- 연속 잭팟 방지: 잭팟 후 200스핀 쿨다운

[운 밸런싱 (Pity System)]
- 1000스핀 무잭팟 시: 잭팟 확률 2배
- 2000스핀 무잭팟 시: 보장 잭팟 지급
```

### 9.4 공격/레이드 밸런싱

```
[공격 빈도]
- 시간당 평균 피공격: 2-3회
- 방패 획득 vs 공격: 1:3 비율 유지
- 24시간 최대 피공격: 10회 (초과분 면역)

[레이드 보호]
- 최소 보유 코인 보호: 500만 (레이드 불가)
- 레이드 쿨다운: 같은 유저에게 4시간 면역
```
