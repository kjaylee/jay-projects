# 버블 디펜스 (Bubble Defense) 기획서

## 📋 기본 정보
- **게임명:** 버블 디펜스 (Bubble Defense)
- **장르:** 하이브리드 캐주얼 (버블 슈터 + 타워 디펜스)
- **플랫폼:** 모바일 웹 / HTML5
- **개발일:** 2026-01-28

---

## 🎯 게임 컨셉 및 배경

### 컨셉
**"쏘고 터뜨려서 타워를 세워라!"**

버블 슈터의 직관적인 조작감과 타워 디펜스의 전략적 깊이를 결합한 하이브리드 캐주얼 게임. 플레이어는 버블을 발사해 매치-3로 터뜨리고, 획득한 에너지로 타워를 건설해 적의 침공을 막는다.

### 배경 스토리
마법의 버블 왕국이 어둠의 슬라임 군단에 침공당했다. 버블 마법사인 플레이어는 색깔 버블을 연결해 에너지를 모으고, 수호 타워를 세워 왕국을 지켜야 한다.

### 2026 트렌드 반영
- **하이브리드 캐주얼:** 하이퍼캐주얼의 접근성 + 미드코어의 전략적 깊이
- **장르 융합:** 버블 슈터(캐주얼) + 타워 디펜스(전략)
- **딥한 프로그레션:** 타워 업그레이드, 스킬 해금, 웨이브 진행

## 👥 타겟 유저

### 1차 타겟
- **연령:** 18-45세
- **성별:** 전체 (버블 슈터는 여성 비율 높음)
- **특징:** 
  - 버블 슈터 경험자 (Bubble Witch, Puzzle Bobble 팬)
  - 타워 디펜스 라이트 유저
  - 출퇴근 시간에 가볍게 즐기는 캐주얼 게이머

### 2차 타겟
- 전략 게임 좋아하지만 복잡한 건 부담스러운 유저
- 퍼즐 게임에 새로운 재미 요소를 원하는 유저

### 유저 페르소나
**"민지, 32세, 직장인"**
- 출퇴근 지하철에서 30분씩 모바일 게임
- 버블 위치 사가 1년 플레이, 최근 지루해짐
- 전략 게임 관심 있지만 복잡한 건 싫음
- "버블 터뜨리는 건 좋은데, 뭔가 더 있으면 좋겠어"

## 🔄 코어 루프

```
┌─────────────────────────────────────────────┐
│                                             │
│  [버블 발사] → [매치-3 터뜨림] → [에너지 획득] │
│       ↑                              ↓      │
│       │                      [타워 건설/강화] │
│       │                              ↓      │
│  [웨이브 클리어] ← [적 처치] ← [타워 공격]   │
│       ↓                                     │
│  [보상 획득] → [다음 웨이브/스테이지]        │
│                                             │
└─────────────────────────────────────────────┘
```

### 세션 구조 (3-5분)
1. **웨이브 시작** (5초) - 적 예고
2. **버블 퍼즐 + 디펜스** (2-4분) - 핵심 플레이
3. **웨이브 클리어** (10초) - 보상/강화 선택
4. **다음 웨이브 or 스테이지 완료**

### 플레이 사이클
- **1 세션:** 5-10 웨이브 (3-5분)
- **1 스테이지:** 3 세션 (10-15분)
- **일일 플레이:** 3-5 스테이지 (30분-1시간)

## 💰 수익화 전략

### 1. 광고 기반 (주력)

#### 보상형 광고
- **추가 타워 슬롯:** 일시적으로 타워 1개 더 설치
- **에너지 부스트:** 버블 매치 시 에너지 2배 (1 웨이브)
- **부활:** 게임 오버 시 체력 50% 회복
- **스핀 휠:** 랜덤 보상 (골드, 버블, 타워 스킨)

#### 배너 광고
- 메인 메뉴 하단
- 스테이지 선택 화면
- 결과 화면

#### 전면 광고
- 스테이지 클리어 시 (3회당 1회)
- 앱 재접속 시

### 2. IAP (In-App Purchase)

#### 광고 제거
- **프리미엄 패스:** $4.99 (광고 제거 + 일일 보석)

#### 소모성 아이템
- **버블 팩:** 특수 버블 (레인보우, 폭탄) 구매
- **골드 팩:** 타워 업그레이드용
- **보석 팩:** 프리미엄 재화

#### 타워 스킨
- 시즌별 테마 스킨 (할로윈, 크리스마스)
- 콜라보 스킨

### 3. 시즌 패스
- 무료 트랙: 기본 보상
- 프리미엄 트랙: $2.99/시즌, 추가 보상 + 전용 스킨

### 수익 예상 분배

- IAP: 30%
- 시즌패스: 10%

## 🎮 핵심 차별점

1. **장르 융합의 신선함:** 버블 슈터 + 타워 디펜스 조합은 시장에 없음
2. **듀얼 스킬 요구:** 퍼즐 실력 + 전략적 배치
3. **능동적 플레이:** 수동적 TD가 아닌 지속적 상호작용
4. **낮은 진입장벽:** 버블 슈터의 직관성 유지
5. **높은 리텐션:** TD의 웨이브 기반 긴장감
