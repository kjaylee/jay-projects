## 2. 슬라임 시스템 상세

### 2.1 슬라임 스탯 구조

```
[기본 스탯]
- HP: 체력 (0이 되면 사망)
- ATK: 공격력
- DEF: 방어력
- SPD: 행동 속도 (턴 순서 결정)
- CRIT: 치명타 확률

[등급별 기본 스탯 범위]
| 등급 | HP | ATK | DEF | SPD |
|------|-----|-----|-----|-----|
| 커먼 | 100-150 | 10-15 | 5-8 | 10-15 |
| 언커먼 | 150-220 | 15-25 | 8-12 | 12-18 |
| 레어 | 220-350 | 25-40 | 12-20 | 15-22 |
| 에픽 | 350-500 | 40-60 | 20-30 | 18-25 |
| 레전더리 | 500-800 | 60-100 | 30-50 | 22-30 |
```

### 2.2 슬라임 속성 상성

```
[상성 배수]
유리: x1.5 데미지
불리: x0.75 데미지
동일: x1.0 데미지

[상성 관계]
🔴 불 → 🟢 풀 (유리)
🟢 풀 → 🔵 물 (유리)
🔵 물 → 🔴 불 (유리)
⚡ 전기 → 🔵 물 (유리)
🌙 어둠 ↔ ☀️ 빛 (상호 유리)
```

### 2.3 슬라임 도감 (100종 예시)

| ID | 이름 | 속성 | 등급 | 스킬 |
|----|------|------|------|------|
| 001 | 파이어젤리 | 불 | 커먼 | 화염구 (단일 30 데미지) |
| 002 | 아쿠아블롭 | 물 | 커먼 | 치유의 물방울 (아군 20 힐) |
| 003 | 리프슬라임 | 풀 | 커먼 | 독침 (3턴 독 상태) |
| 010 | 썬더푸딩 | 전기 | 언커먼 | 연쇄 번개 (2체 공격) |
| 020 | 쉐도우젤 | 어둠 | 레어 | 그림자 일격 (치명타율 +30%) |
| 050 | 피닉스슬라임 | 불 | 에픽 | 부활 (1회 사망 시 50% HP 부활) |
| 100 | 레인보우슬라임 | 전속성 | 레전 | 프리즘 버스트 (전체 100 데미지) |

### 2.4 합성 시스템 상세

```
[기본 합성]
동일 슬라임 3개 → 상위 등급 1개 (랜덤)

[합성 확률]
같은 종류 3개 합성 시:
- 같은 종류 상위 등급: 60%
- 랜덤 상위 등급: 40%

[특수 레시피 (확정)]
파이어젤리 + 아쿠아블롭 + 리프슬라임 = 레인보우슬라임
썬더푸딩 + 썬더푸딩 + 썬더푸딩 = 스톰킹슬라임
쉐도우젤 + 빛슬라임 + 빛슬라임 = 이클립스슬라임

[합성 비용]
커먼→언커먼: 100 골드
언커먼→레어: 300 골드
레어→에픽: 1000 골드
에픽→레전더리: 5000 골드
```

## 3. 전투 시스템 상세

### 3.1 배치 그리드

```
[3x3 그리드 포지션 효과]
┌─────┬─────┬─────┐
│ 전좌 │ 전중 │ 전우 │  ← 앞줄: ATK +20%, DEF -10%
├─────┼─────┼─────┤
│ 중좌 │ 중중 │ 중우 │  ← 중간: 스킬 쿨다운 -1턴
├─────┼─────┼─────┤
│ 후좌 │ 후중 │ 후우 │  ← 뒷줄: DEF +20%, ATK -10%
└─────┴─────┴─────┘

[특수 포지션]
중중 (정중앙): 보스 타겟 우선순위 ↑
전중: 도발 효과 (적 공격 집중)
후좌/후우: 회피율 +10%
```

### 3.2 전투 흐름

```
[턴 진행]
1. 속도(SPD) 순으로 행동 순서 결정
2. 각 슬라임이 자동으로 스킬 사용
3. 타겟 선택: 기본 = 가장 가까운 적
4. 데미지 계산 → HP 감소
5. 사망 체크 (HP ≤ 0)
6. 승리/패배 판정

[데미지 공식]
최종 데미지 = (ATK × 스킬배수 × 상성배수) - DEF
최소 데미지 = 1 (보장)
치명타 = 최종 데미지 × 1.5
```

### 3.3 스킬 시스템

```
[스킬 타입]
- 단일 공격: 한 대상에게 데미지
- 범위 공격: 여러 대상에게 데미지
- 힐: 아군 HP 회복
- 버프: 아군 스탯 증가
- 디버프: 적 스탯 감소
- 패시브: 상시 적용 효과

[쿨다운]
스킬마다 쿨다운 존재 (0-3턴)
쿨다운 중에는 기본 공격 사용
```

## 4. 런 구조 상세

### 4.1 스테이지 구성

```
[1런 = 10스테이지]

스테이지 1: 슬라임 x3 (커먼)
스테이지 2: 슬라임 x4 (커먼)
스테이지 3: 슬라임 x5 (커먼+언커먼) → 미니 보상
스테이지 4: 슬라임 x5 (언커먼)
스테이지 5: 미니보스 + 슬라임 x2 → 보상 선택
스테이지 6: 슬라임 x6 (레어 포함)
스테이지 7: 슬라임 x6 (레어 다수)
스테이지 8: 슬라임 x7 → 보상 선택
스테이지 9: 엘리트 슬라임 x3 (에픽)
스테이지 10: 보스 1체 → 런 완료 보상
```

### 4.2 보상 시스템

```
[스테이지 클리어 보상 (3택 1)]
- 슬라임: 새로운 슬라임 획득
- 장비: 스탯 보너스 장비
- 재화: 골드 또는 젬

[런 완료 보상]
- 영구 골드: 스테이지 수 × 100
- 영구 젬: 클리어 시 50
- 도감 등록: 새로운 슬라임 해금
```

### 4.3 난이도 스케일링

```
[스테이지별 적 스탯 배수]
스테이지 1: x1.0
스테이지 2: x1.1
스테이지 3: x1.2
...
스테이지 10 (보스): x2.5

[무한 모드 (언락 후)]
스테이지 11+: 매 스테이지 x1.15 증가
리더보드 등록
```

## 5. 메타 진행 시스템

### 5.1 영구 업그레이드

```
[골드 업그레이드]
- 시작 슬라임 +1 (최대 +2): 1000/5000 골드
- 기본 HP +10%: 500 골드 (10단계)
- 기본 ATK +5%: 500 골드 (10단계)
- 합성 비용 -10%: 800 골드 (5단계)

[젬 업그레이드]
- 레어 슬라임 출현율 +5%: 100 젬 (5단계)
- 에픽 슬라임 출현율 +2%: 200 젬 (5단계)
- 추가 보상 선택지 +1: 500 젬
```

### 5.2 도감 보상

```
[수집률 보상]
10종 수집: 골드 +10% 보너스
25종 수집: 시작 장비 1개
50종 수집: 레어 슬라임 보장 티켓
75종 수집: 에픽 슬라임 보장 티켓
100종 수집: 레전더리 슬라임 선택권
```

## 7. 밸런싱 수치

### 7.1 경제 밸런싱

```
[1런 평균 획득]
- 골드: 800-1500 (클리어 기준)
- 젬: 50-100 (클리어 + 광고)
- 슬라임: 5-8마리

[지출 구조]
- 합성 1회: 100-5000 골드
- 영구 업그레이드: 500-5000 골드
- 가챠 1회: 100 젬
```

### 7.2 진행 속도

```
[예상 진행]
- 첫 클리어: 3-5런 (30-50분)
- 무한모드 해금: 10런 (2시간)
- 도감 50%: 50런 (1주일)
- 도감 100%: 200런 (1개월)
```
