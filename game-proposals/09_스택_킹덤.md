# 🏗️ 스택 킹덤 (Stack Kingdom)

## 📋 기본 정보
- **장르**: 스태킹 + 왕국 건설
- **플랫폼**: iOS / Android
- **타겟**: 전연령 캐주얼
- **예상 개발기간**: 3-4개월
- **팀 규모**: 2-3명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "블록을 쌓아 왕국을 건설! 정확히 쌓을수록 더 웅장한 성이 완성되는 스태킹 빌더"

### 핵심 재미 요소
1. **스태킹 긴장감** - 타이밍 맞추기
2. **시각적 보상** - 성이 점점 커짐
3. **왕국 꾸미기** - 건물/데코 배치
4. **경쟁 요소** - 가장 높은 탑

---

## ⚙️ 핵심 메카닉

### 1. 스태킹 시스템
```
[기본 메카닉]
블록이 좌우로 이동
탭하면 현재 위치에 드랍
아래 블록에 정확히 맞춰야 함

[판정]
- Perfect: 100% 유지 + 보너스
- Good: 90% 이상 유지
- Okay: 70% 이상 유지
- 70% 미만: 삐져나온 부분 탈락

[콤보 시스템]
연속 Perfect → 콤보 보너스
5 Perfect 연속 → 블록 확장!
```

### 2. 건물 진화
```
[층수별 변화]
1-10층: 농가 🏠
11-20층: 저택 🏛️
21-30층: 탑 🗼
31-40층: 성 🏰
41-50층: 대성당 ⛪
51층+: 천상의 성 ☁️🏰

[시각 변화]
층이 쌓일수록 건물 디테일 증가
10층마다 건물 스타일 진화
```

### 3. 왕국 시스템
```
[왕국 맵]
완성된 건물들을 왕국에 배치
여러 건물로 왕국 구성

[건물 타입]
🏠 주거: 인구 ↑
🏛️ 관청: 골드 생산
⛪ 신전: 버프 효과
🗼 탑: 방어력
🏰 성: 왕국 레벨

[배치 보너스]
인접 건물끼리 시너지
예: 주거 + 관청 = 세금 수입 ↑
```

### 4. 자원 시스템
```
[재화]
🪙 골드: 스태킹 보상
💎 젬: 프리미엄 재화
🧱 벽돌: 건물 강화

[사용처]
- 건물 강화
- 데코 구매
- 왕국 확장
```

### 5. 챌린지 모드
```
[일일 챌린지]
매일 다른 조건의 스태킹
- 속도 증가
- 좁은 블록
- 바람 효과
- 시간 제한

[주간 토너먼트]
전 세계 유저와 경쟁
가장 높은 탑 = 1위
```
