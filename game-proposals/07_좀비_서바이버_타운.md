# 🧟 좀비 서바이버 타운 (Zombie Survivor Town)

## 📋 기본 정보
- **장르**: 뱀서라이크 + 기지 건설
- **플랫폼**: iOS / Android
- **타겟**: 미드코어 (18-40세 남성)
- **예상 개발기간**: 5-6개월
- **팀 규모**: 3-5명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "낮에는 마을을 건설하고, 밤에는 좀비와 싸워라! 뱀서라이크 전투 + 서바이벌 기지 건설"

### 핵심 재미 요소
1. **뱀서라이크 전투** - 밤마다 웨이브 서바이벌
2. **기지 건설** - 낮에 마을 업그레이드
3. **자원 관리** - 식량, 재료, 생존자
4. **생존자 구출** - 동료 수집 & 배치

---

## ⚙️ 핵심 메카닉

### 1. 낮/밤 사이클
```
[낮 페이즈] (3-5분)
- 기지 건설 & 업그레이드
- 자원 수집 (탐험)
- 생존자 관리
- 장비 제작

[밤 페이즈] (5-10분)
- 뱀서라이크 전투
- 좀비 웨이브 방어
- 보스 출현 (특정 밤)
- 아이템 드랍 수집
```

### 2. 뱀서라이크 전투
```
[조작]
- 가상 조이스틱: 이동
- 공격: 자동 (가장 가까운 적)

[레벨업 시스템]
경험치 획득 → 레벨업 → 3개 중 1개 선택

[무기/스킬 예시]
🔫 권총: 기본 단발
🔪 나이프 서클: 회전 근접
💣 수류탄: 범위 폭발
⚡ 전기 충격: 연쇄 공격
🛡️ 보호막: 피격 무효
❤️ 회복 오라: 지속 힐

[진화 시스템]
같은 무기 선택 → 레벨업 (최대 Lv.5)
Lv.5 + 특정 아이템 = 진화 무기
예: 권총 Lv.5 + 조준경 = 스나이퍼
```

### 3. 기지 건설
```
[건물 종류]
🏠 주거지: 생존자 수용 (+인구)
🌾 농장: 식량 생산
⚒️ 대장간: 무기/방어구 제작
🏥 병원: 부상자 치료
🔬 연구소: 기술 연구
🏰 방벽: 밤 전투 버프

[업그레이드]
건물 레벨업 → 효율 증가
예: 농장 Lv.3 → 식량 생산 +50%
```

### 4. 자원 시스템
```
[자원 종류]
🍖 식량: 생존자 유지
🪵 목재: 건물 건설
🔩 철: 무기 제작
💎 희귀 재료: 고급 장비

[획득 방법]
- 건물 생산 (자동)
- 낮 탐험 (미니게임)
- 밤 전투 드랍
```

### 5. 생존자 시스템
```
[역할]
⚔️ 전사: 밤 전투 참여
👷 일꾼: 건물 건설 속도 ↑
🔬 연구원: 연구 속도 ↑
🌾 농부: 식량 생산 ↑
🏥 의사: 치료 효율 ↑

[생존자 획득]
- 밤 전투 후 구출 이벤트
- 탐험 중 발견
- 특별 이벤트
```
