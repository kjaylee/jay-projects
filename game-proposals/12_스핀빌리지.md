# 🎰 스핀빌리지 (Spin Village)

## 📋 기본 정보
- **장르**: 소셜 슬롯 빌더 (Slot Machine + Village Builder)
- **플랫폼**: iOS / Android (세로모드)
- **타겟**: 캐주얼~미드코어 (25-55세, 특히 여성 유저 40%+)
- **예상 개발기간**: 4-6개월
- **팀 규모**: 2-4명
- **영감**: Coin Master, Piggy GO, Island King

---

## 🎮 게임 컨셉

### 한 줄 요약
> "슬롯을 돌려 코인과 공격권을 얻고, 나만의 마을을 건설하며 친구를 레이드하라!"

### 핵심 아이디어
Coin Master의 검증된 코어 루프를 기반으로, 더 풍부한 마을 건설 콘텐츠와 소셜 인터랙션을 추가합니다.

### 핵심 재미 요소
1. **슬롯의 도파민** - 릴이 멈추는 순간의 기대감과 잭팟 터지는 쾌감
2. **건설의 성취감** - 마을이 점점 화려해지는 시각적 만족
3. **소셜 복수극** - 친구 마을 공격/레이드의 경쟁심
4. **수집의 재미** - 카드 컬렉션, 펫 수집
5. **진행 동력** - 다음 마을 테마를 향한 목표의식

## ⚙️ 핵심 메카닉

### 1. 슬롯 시스템

```
[릴 구성: 3x1 (3개 슬롯)]

심볼 종류:
🪙 코인 - 보유 코인 증가
⚔️ 해머 - 상대 마을 건물 공격
🛡️ 방패 - 공격으로부터 보호
💰 돼지 저금통 - 상대 코인 레이드
👑 왕관 - 보너스 스핀 (3개 일치 시 잭팟)
⭐ 스캐터 - 프리스핀 진입
```

### 2. 베팅 시스템
| 베팅 레벨 | 스핀당 비용 | 보상 배수 |
|-----------|-------------|-----------|
| x1 | 100만 코인 | x1 |
| x2 | 200만 코인 | x2 |
| x5 | 500만 코인 | x5 |
| x10 | 1000만 코인 | x10 |

### 3. 마을 건설 시스템

```
[마을 구조]
- 마을 1개 = 건물 5개
- 건물당 5레벨 업그레이드
- 모든 건물 5레벨 달성 → 다음 마을 이동

[건물 업그레이드 비용 예시 - 마을 1]
레벨 1: 50만 코인
레벨 2: 100만 코인
레벨 3: 200만 코인
레벨 4: 500만 코인
레벨 5: 1000만 코인
```

### 4. 공격/방어 시스템

**공격 (해머 3개 일치)**
- 랜덤 또는 친구 마을 선택
- 상대 건물 1개에 데미지 (레벨 -1)
- 방패가 있으면 방패 소모, 건물 보호

**방패 시스템**
- 최대 3개 보유 가능
- 공격 1회당 방패 1개 소모
- 방패 없으면 건물 직접 피해

### 5. 레이드 시스템

**레이드 (돼지 3개 일치)**
- 상대 마을의 숨겨진 코인 위치 선택 (3개 중 1개)
- 성공 시 상대 코인의 일부 획득
- 펫 보너스로 레이드 배수 증가

## 👥 타겟 유저

### 1차 타겟: 캐주얼 소셜 게이머
- **연령**: 30-50세
- **성별**: 여성 60%, 남성 40%
- **특성**: 가볍게 즐기면서 친구와 소통하고 싶은 유저
- **플레이 패턴**: 출퇴근 시간, 점심시간, 취침 전

### 2차 타겟: 슬롯/카지노 게임 팬
- **연령**: 25-45세
- **특성**: 슬롯의 도파민을 합법적으로 즐기고 싶은 유저
- **플레이 패턴**: 자투리 시간 수시 접속

### 유저 페르소나
```
"지영" (38세, 직장인 여성)
- Coin Master 1년 넘게 플레이
- 회사 동료들과 같이 플레이, 서로 공격하며 재미
- 하루 3-4회 접속, 회당 5-10분
- 월 $5-10 정도 소액 과금
```

## ✅ 재미 분석

### 장점 (Pros)
- ✅ **검증된 모델**: Coin Master가 연 매출 $1B+ 증명
- ✅ **단순한 조작**: 원탭 게임, 누구나 쉽게
- ✅ **강력한 바이럴**: 친구 공격/레이드가 자연스러운 초대 유도
- ✅ **고착률 높음**: 공격당하면 복수하러 재접속
- ✅ **긴 수명**: 200+ 마을로 장기 플레이
- ✅ **수익성 검증**: 슬롯+소셜 조합의 높은 ARPU

### 단점 (Cons)
- ⚠️ **경쟁 치열**: Coin Master가 시장 장악
- ⚠️ **차별화 필요**: 독자적 요소 없으면 카피캣 취급
- ⚠️ **UA 비용**: 캐주얼 장르 CPI 높음
- ⚠️ **사행성 이슈**: 일부 국가 규제 주의

### 차별화 전략
1. **테마 다양화**: 한국/아시아 특화 마을 테마
2. **펫 컨텐츠 강화**: 펫 육성/합성/진화 시스템
3. **길드 시스템**: 클랜 레이드 보스 도입
4. **미니게임**: 슬롯 외 보너스 미니게임
