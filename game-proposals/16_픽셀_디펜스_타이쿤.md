## ⚙️ 핵심 메카닉

### 1. 유닛 시스템
```
[유닛 종류] (Tiny Swords 기반)
- 🗡️ 검사: 근접, 균형잡힌 스탯
- 🏹 궁수: 원거리, 낮은 체력
- 🛡️ 기사: 근접, 높은 체력
- 🧙 마법사: 범위 공격
- 👑 영웅: 강력한 특수 유닛

[유닛 강화]
- 레벨업: 골드 소모
- 등급업: 같은 유닛 3개 합성
- 장비: 무기/방어구 장착
```

### 2. 전투 시스템
```
[웨이브 구조]
- 각 스테이지 10웨이브
- 웨이브마다 적 수/강도 증가
- 보스 웨이브 (5, 10)

[유닛 배치]
- 3x5 그리드
- 앞줄: 탱커
- 중간: 근접 딜러
- 뒷줄: 원거리

[자동 전투]
- 배치 후 자동 진행
- 배속 조절 (1x, 2x, 4x)
```

### 3. 왕국 건설
```
[건물 종류]
- 🏠 집: 인구 증가 → 유닛 슬롯
- ⚒️ 대장간: 무기 제작
- 🏫 훈련소: 유닛 경험치 부스트
- 💰 금광: 골드 생산 (아이들)
- 🌾 농장: 식량 생산 (아이들)

[아이들 수익]
- 금광/농장에서 시간당 자원 생산
- 오프라인 최대 8시간 축적
- 광고 시청 시 2배 수령
```

### 4. 진행 구조
```
[월드 맵]
- 챕터 1: 초원 (스테이지 1-20)
- 챕터 2: 숲 (스테이지 21-40)
- 챕터 3: 사막 (스테이지 41-60)
- 챕터 4: 설산 (스테이지 61-80)
- 챕터 5: 마왕성 (스테이지 81-100)

[별 수집]
- 스테이지 클리어 시 별 1-3개
- 별로 왕국 건물 해금
```
