## 2. 뱀서라이크 전투 시스템 상세

### 2.1 기본 조작

```
[조작]
가상 조이스틱: 좌측 하단 (이동)
공격: 자동 (가장 가까운 적)
스킬: 자동 발동 (쿨다운 후)

[카메라]
탑다운 고정 시점
플레이어 중앙 배치
줌: 80-120% (무기 범위에 따라)
```

### 2.2 데미지 공식

```
[기본 데미지 계산]
최종 데미지 = (무기 ATK × 스킬 배수) × (1 + 버프%) - 적 DEF
최소 데미지 = 1 (보장)

치명타 = 최종 데미지 × (1.5 + 치명타 배수)
치명타 확률 = 5% + 장비/스킬 보너스

[DPS 예시]
권총: ATK 10 × 초당 2발 = DPS 20
샷건: ATK 30 × 초당 0.5발 = DPS 15 (but 범위)
```

### 2.3 무기 시스템

```
[무기 종류 - 기본 8종]

🔫 권총 (Pistol)
  - 타입: 단일 타겟
  - ATK: 10 | 속도: 2/초
  - 특징: 밸런스형 기본 무기

🔪 나이프 서클 (Knife Circle)
  - 타입: 근접 회전
  - ATK: 15 | 회전 속도: 1회/초
  - 특징: 칼이 플레이어 주변 회전

💣 수류탄 (Grenade)
  - 타입: 범위 폭발
  - ATK: 50 | 쿨다운: 3초
  - 특징: 중심부 데미지 최대

⚡ 전기 충격 (Electric Shock)
  - 타입: 연쇄 공격
  - ATK: 8 | 연쇄: 3체
  - 특징: 적이 몰려있을 때 효과적

🛡️ 보호막 (Shield)
  - 타입: 방어
  - 흡수: 30 데미지 | 쿨다운: 10초
  - 특징: 피격 무효화

❤️ 회복 오라 (Heal Aura)
  - 타입: 지속 힐
  - 회복: 5 HP/초
  - 특징: 생존력 증가

🔥 화염방사기 (Flamethrower)
  - 타입: 전방 범위
  - ATK: 5/틱 | 지속: 2초
  - 특징: 지속 데미지

❄️ 냉기 폭풍 (Frost Storm)
  - 타입: 전체 슬로우
  - 효과: 이동속도 -30%
  - 특징: 컨트롤 유틸

[무기 레벨업]
같은 무기 선택 시 레벨업 (최대 Lv.5)
Lv별 강화: ATK +20%, 특수효과 증가
```

### 2.4 무기 진화 시스템

```
[진화 조건]
무기 Lv.5 + 특정 패시브 아이템 = 진화 무기

[진화 테이블]
| 무기 | 필요 아이템 | 진화 무기 | 효과 |
|------|-----------|----------|------|
| 권총 | 조준경 | 스나이퍼 | 관통 + 크리티컬 |
| 나이프 | 속도부츠 | 블레이드 댄스 | 칼 수 2배 |
| 수류탄 | 화약통 | 클러스터 폭탄 | 자탄 생성 |
| 전기충격 | 배터리 | 썬더스톰 | 연쇄 무제한 |
| 보호막 | 가시장갑 | 반사막 | 피격 시 반사 |
| 회복오라 | 생명의 보석 | 불사의 오라 | 사망 1회 방지 |
| 화염방사 | 기름통 | 지옥불 | 장판 생성 |
| 냉기폭풍 | 빙결석 | 얼음 감옥 | 일정 확률 동결 |
```

### 2.5 경험치 & 레벨업


## 5. 자원 시스템 상세

### 5.1 자원 종류

```
[기본 자원]
🍖 식량 (Food)
  - 용도: 생존자 유지 (인당 10/일)
  - 획득: 농장, 탐험, 밤 전투
  - 저장 한도: 500 (창고 레벨당 +200)

🪵 목재 (Wood)
  - 용도: 건물 건설/업그레이드
  - 획득: 제재소, 탐험
  - 저장 한도: 300

🔩 철 (Iron)
  - 용도: 무기 제작, 고급 건물
  - 획득: 광산, 밤 전투
  - 저장 한도: 200

💎 희귀 재료 (Rare)
  - 용도: 고급 장비, 연구
  - 획득: 보스 처치, 특수 이벤트
  - 저장 한도: 50
```

### 5.2 자원 순환

```
[일일 자원 흐름]

수입:
- 농장 Lv3: +30 식량/시간
- 탐험 1회: +50 목재, +20 철
- 밤 전투 클리어: +100 식량, +50 목재, +30 철

지출:
- 생존자 5명: -50 식량/일
- 건물 유지비: 없음
- 업그레이드: 변동

[밸런스 포인트]
농장 Lv3 + 생존자 5명 = 식량 자급자족
추가 생존자 = 탐험/전투 필요
```

```
  - 기본 스탯: 연구속도 100%

  - 특수: 배치당 속도 +15%

🌾 농부 (Farmer)
  - 기본 스탯: 생산량 100%
  - 역할: 식량 생산 증가
  - 특수: 농장당 +20% 생산

🏥 의사 (Medic)
  - 기본 스탯: 치료속도 100%
```

```csharp
[플레이어 컨트롤러]
public class SurvivorController : MonoBehaviour {
    private Rigidbody2D rb;
    private List<Weapon> weapons = new();
    
    void Update() {
        // 이동 (조이스틱 입력)
        Vector2 input = joystick.Direction;
        rb.velocity = input * moveSpeed;
        
        // 자동 공격
        foreach (var weapon in weapons) {
            if (weapon.CanFire()) {
                Enemy target = FindClosestEnemy();
                if (target != null) {
                    weapon.Fire(target.transform.position);
                }
            }
        }
    }
}

[무기 시스템]
public abstract class Weapon : MonoBehaviour {
    public int level = 1;
    public float baseDamage;
    public float cooldown;
    private float lastFireTime;
    
    public bool CanFire() => Time.time >= lastFireTime + cooldown;
    
    public virtual void Fire(Vector3 target) {
        lastFireTime = Time.time;
        SpawnProjectile(target);
        PlayFireEffect();
    }
    
    public virtual float GetDamage() {
        return baseDamage * (1 + (level - 1) * 0.2f);
    }
}
```

```csharp
[좀비 AI]
public class ZombieAI : MonoBehaviour {
    private Transform player;
    private float speed;
    private float attackRange;
    
    void Update() {
        // 플레이어 추적
        Vector2 direction = (player.position - transform.position).normalized;
        transform.position += (Vector3)direction * speed * Time.deltaTime;
        
        // 공격 범위 체크
        float distance = Vector2.Distance(transform.position, player.position);
        if (distance <= attackRange) {
            Attack();
        }
    }
    
    private void Attack() {
        if (Time.time >= lastAttackTime + attackCooldown) {
            player.GetComponent<SurvivorController>().TakeDamage(attackDamage);
            lastAttackTime = Time.time;
        }
    }
}

[스포너 시스템]
public class ZombieSpawner : MonoBehaviour {
    public void SpawnWave(WaveData wave) {
        foreach (var spawnInfo in wave.spawns) {
            StartCoroutine(SpawnOverTime(
                spawnInfo.prefab, 
                spawnInfo.count, 
                spawnInfo.interval
            ));
        }
    }
    
    private IEnumerator SpawnOverTime(GameObject prefab, int count, float interval) {
        for (int i = 0; i < count; i++) {
            Vector2 spawnPos = GetRandomSpawnPosition();
            Instantiate(prefab, spawnPos, Quaternion.identity);
            yield return new WaitForSeconds(interval);
        }
    }
}
```

```csharp
public class BuildingPlacer : MonoBehaviour {
    private Building selectedBuilding;
    private Grid grid;
    
    public void StartPlacement(BuildingData data) {
        selectedBuilding = Instantiate(data.ghostPrefab);
        selectedBuilding.SetGhostMode(true);
    }
    
    void Update() {
        if (selectedBuilding != null) {
            Vector3 worldPos = Camera.main.ScreenToWorldPoint(Input.mousePosition);
            Vector3Int gridPos = grid.WorldToCell(worldPos);
            selectedBuilding.transform.position = grid.CellToWorld(gridPos);
            
            bool canPlace = CanPlaceAt(gridPos, selectedBuilding.size);
            selectedBuilding.SetValidPlacement(canPlace);
            
            if (Input.GetMouseButtonDown(0) && canPlace) {
                PlaceBuilding(gridPos);
            }
        }
    }
    
    private void PlaceBuilding(Vector3Int gridPos) {
        Building placed = Instantiate(selectedBuilding.dataPrefab, 
            grid.CellToWorld(gridPos), Quaternion.identity);
        placed.Initialize();
        MarkGridOccupied(gridPos, placed.size);
        selectedBuilding = null;
    }
}
```

```csharp
[게임 사이클 매니저]
public class DayNightCycle : MonoBehaviour {
    public enum Phase { Day, Night }
    public Phase currentPhase = Phase.Day;
    
    public float dayDuration = 300f; // 5분
    public float nightDuration = 600f; // 10분
    
    private float phaseTimer;
    
    void Update() {
        phaseTimer -= Time.deltaTime;
        
        if (phaseTimer <= 0) {
            SwitchPhase();
        }
        
        UI.UpdateTimer(phaseTimer);
    }
    
    private void SwitchPhase() {
        if (currentPhase == Phase.Day) {
            currentPhase = Phase.Night;
            phaseTimer = nightDuration;
            StartNightBattle();
        } else {
            currentPhase = Phase.Day;
            phaseTimer = dayDuration;
            EndNightBattle();
        }
    }
}
```
