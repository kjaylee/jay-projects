### 게임 시스템 상세

#### 전투/메인 메카닉

**1. 삼중 접근 시스템 (Triple Approach)**
| 접근법 | 설명 | 장점 | 단점 |
|--------|------|------|------|
| **고스트** | 완전 스텔스, 흔적 없음 | 보너스 보상, 평판 유지 | 시간 소요, 높은 난이도 |
| **프레데터** | 스텔스 + 무력화 | 균형잡힌 접근 | 일부 흔적 남음 |
| **스톰** | 전면 전투 돌파 | 빠른 진행 | 평판 하락, 자원 소모 |

**2. 해킹 미니게임**
- 노드 연결 퍼즐 (Pipes 스타일)
- 시간 제한 + 보안 레벨에 따른 난이도
- 성공 시: 카메라 끄기, 문 열기, 정보 탈취
- 실패 시: 경보 발동, 증원 호출

**3. 탐정 시스템**
- 단서 수집 → 증거판에 배치 → 연결선 긋기
- 올바른 연결 = 진실 발견
- 잘못된 연결 = 오판 (다른 결말로 분기)

#### 성장 시스템

**1. 사이버웨어 슬롯 (6개)**
| 슬롯 | 예시 업그레이드 |
|------|-----------------|
| **눈** | 야간 시야, 적 아웃라인, 약점 표시 |
| **뇌** | 해킹 속도 +, 시간 슬로우 |
| **팔** | 무기 반동 감소, 근접 데미지 + |
| **다리** | 이동 속도, 점프 높이, 조용한 발걸음 |
| **척추** | 체력 회복, 방어력 |
| **스킨** | 위장, 충격 흡수 |

**2. 스킬 트리 (3갈래)**
- 🔴 **전투**: 무기 숙련, 치명타, 체력
- 🟢 **잠입**: 스텔스, 해킹, 함정
- 🔵 **사회**: 대화, 정보망, 평판 보너스

**3. 레벨업 조건**
- XP = 미션 완료 + 선택적 목표 달성
- 레벨당: 스킬 포인트 2개 + 사이버웨어 용량 증가

#### 경제 시스템

**화폐 종류**
| 화폐 | 획득처 | 용도 |
|------|--------|------|
| **크레딧** | 미션 보상, 아이템 판매 | 장비/무기 구매, 정보 매매 |
| **암호 코인** | 해킹, 숨겨진 금고 | 사이버웨어, 희귀 아이템 |
| **평판 토큰** | 팩션 미션 완료 | 팩션 전용 장비 해금 |

**경제 밸런스**
- 미션 보상: 1,000 ~ 5,000 크레딧
- 기본 무기: 2,000 크레딧
- 고급 사이버웨어: 10,000 크레딧 + 5 암호 코인
- 인플레이션 방지: 유지비 (아파트 월세, 사이버웨어 관리)

#### 소셜/멀티플레이

**싱글 중심 + 비동기 소셜**
- **정보 공유**: 다른 플레이어의 단서 힌트 열람 (유료)
- **고스트 리플레이**: 다른 플레이어의 잠입 경로 유령으로 표시
- **리더보드**: 에피소드별 스피드런 / 고스트 클리어 랭킹
- **시즌 이벤트**: 협동 목표 (전체 플레이어 기업 해킹 횟수 등)

## 🎯 재미 분석

### 재미 요소 (Fun Factors)

| 요소 | 점수 | 설명 |
|------|:----:|------|
| 즉각적 피드백 | ⭐⭐⭐ (3) | 해킹/잠입 성공 시 만족감 있으나, 탐정 파트는 느린 편 |
| 성장 만족감 | ⭐⭐⭐⭐ (4) | 사이버웨어로 능력 변화가 체감됨, 스토리 진행 만족 |
| 전략적 깊이 | ⭐⭐⭐⭐⭐ (5) | 삼중 접근 시스템 + 선택에 따른 분기 = 높은 리플레이성 |
| 수집 욕구 | ⭐⭐⭐ (3) | 사이버웨어, 무기 수집 가능하나 핵심 재미는 아님 |
| 사회적 재미 | ⭐⭐ (2) | 싱글 중심, 비동기 소셜은 제한적 |
| 스토리/몰입 | ⭐⭐⭐⭐⭐ (5) | 사이버펑크 세계관 + 탐정 미스터리 = 강력한 몰입 |

**종합 점수: 22/30 (73%)**

### ✅ 장점

1. **차별화된 장르 믹스**
   - 잠입 + 탐정 + RPG의 독특한 조합
   - 시장에 유사 게임 부재 (틈새 시장)

2. **높은 리플레이 가치**
   - 3가지 접근법 × 선택 분기 = 다회차 플레이 유도
   - 에피소드별 다른 결말

3. **에셋 활용 극대화**
   - 4.3GB Cyberpunk City 에셋 완전 활용
   - 비주얼만으로 경쟁력 확보

4. **스토리텔링 강점**
   - 에피소드 구조로 드라마 느낌
   - 시즌제 업데이트로 장기 운영 가능

### ⚠️ 단점 / 리스크

1. **개발 복잡도**
   - 분기 시스템 = QA 어려움
   - 탐정 시스템 설계 난이도 높음

2. **긴 개발 기간**
   - 8-10개월 = 자금/동기 유지 리스크
   - MVP 테스트까지 시간 소요

3. **타겟 시장 불확실성**
   - 사이버펑크 피로감 (2077 이후)
   - 탐정 게임은 니치 장르

4. **모바일 전환 어려움**
   - 복잡한 UI/UX = 터치 조작 한계
   - PC 중심 → 시장 규모 제한

### 🎮 타겟 유저 심리

**이런 사람이 좋아합니다:**
- 💜 Deus Ex, Dishonored 팬 (잠입 + 선택의 자유)
- 💜 Disco Elysium, LA Noire 팬 (탐정/추리)
- 💜 사이버펑크 세계관에 빠진 사람
- 💜 스토리 중심 게임을 선호하는 성인 게이머

**핵심 동기:**
- **Immersion** (몰입): 사이버펑크 세계에 빠지고 싶다
- **Mastery** (숙달): 완벽한 잠입/추리를 달성하고 싶다
- **Narrative** (서사): 내 선택으로 스토리를 만들고 싶다

# 2️⃣ 하이스트 크루 (Heist Crew)
> "로우폴리 스타일의 턴제 전략 강도 시뮬레이션"

---

## 📋 상세 설계서

### 코어 루프

```
┌─────────────────────────────────────────────────────────────────┐
│                    [ 본부 - 메인 허브 ]                          │
│         ↙              ↓              ↓              ↘          │
│    히스트 선택     크루 관리      장비 구매       스킬 업그레이드   │
│         ↘              ↓              ↓              ↙          │
│                  [ 작전 계획 단계 ]                               │
│                        ↓                                        │
│    ┌─────────────────────────────────────────────────┐         │
│    │  청사진 분석 → 진입로 선택 → 팀원 배치 → 타이밍 설정  │         │
│    └─────────────────────────────────────────────────┘         │
│                        ↓                                        │
│                  [ 실행 단계 ]                                   │
│    ┌─────────────────────────────────────────────────┐         │

│    └─────────────────────────────────────────────────┘         │
│                        ↓                                        │
│              ┌────────┴────────┐                               │
│              ↓                  ↓                               │

│           ↓                    ↓                                │
│      보상 + 명성            크루 체포/부상                       │
│      + 리플레이 영상         + 재시도 옵션                        │
│              ↘                ↙                                │
│                [ 본부 복귀 ]                                     │
│                     ↓                                           │
│            다음 히스트 해금 / 명성 상승                           │
└─────────────────────────────────────────────────────────────────┘
```

### 게임 시스템 상세

#### 전투/메인 메카닉

**1. 계획 단계 (턴제)**

| 단계 | 설명 | 시간 |
|------|------|------|
| **청사진 분석** | 맵 구조, 보안 카메라, 경비 패턴 확인 | 무제한 |
| **진입로 선택** | 정문/후문/환기구/하수구 등 | - |
| **팀원 배치** | 각 역할에 맞는 위치 지정 | - |
| **동기화** | 타이밍 설정 (00초에 A가 해킹, 05초에 B 진입) | - |

**2. 실행 단계 (실시간 + 일시정지)**
- 계획대로 자동 진행
- 돌발 상황 발생 시 일시정지 + 선택
  - 예: "경비가 예상보다 일찍 복귀! A를 숨길까, B를 보낼까?"
- 크리티컬 모먼트: 슬로우모션 + 판정 (성공/실패 확률 표시)

**3. 역할 시스템**

| 역할 | 능력 | 필수 장면 |
|------|------|----------|
| 🔓 **록스미스** | 금고/자물쇠 해제 | 금고실 |
| 💻 **해커** | 카메라/보안 비활성화 | 보안실 |
| 💪 **근육** | 경비 제압, 무거운 물건 | 돌발 상황 |
| 🚗 **드라이버** | 탈출 차량, 도주 경로 | 탈출 |
| 🎭 **페이스** | 변장, 사회공학 | 내부 침투 |
| 👀 **워처** | 외부 감시, 경고 | 전 과정 |

**4. 돌발 상황 시스템**
- 매 히스트마다 1-3개 랜덤 이벤트
- 플레이어 선택에 따라 결과 분기
- 같은 히스트도 매번 다른 전개

#### 성장 시스템

**1. 크루 레벨업**
- 경험치: 히스트 참여 + 역할 수행
