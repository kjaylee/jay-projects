## 2. 카드 시스템 상세

### 2.1 카드 풀

```
[카드 수]
총 카드: 150장
- 공격: 50장
- 방어: 30장
- 마법: 40장
- 회복: 15장
- 버프: 15장

[등급 분포]
커먼 (흰): 60% (90장)
언커먼 (초록): 25% (38장)
레어 (파랑): 10% (15장)
에픽 (보라): 4% (6장)
레전더리 (금): 1% (1장)
```

### 2.2 카드 예시

```
[공격 카드]
⚔️ 베기 (커먼): 10 데미지
⚔️ 강타 (언커먼): 20 데미지
⚔️ 연속베기 (레어): 8 x 3 데미지
⚔️ 처형 (에픽): 적 HP 20% 이하 즉사
⚔️ 오버드라이브 (레전): 이번 턴 데미지 x3

[방어 카드]
🛡️ 방어 (커먼): 5 실드
🛡️ 철벽 (언커먼): 10 실드
🛡️ 반격 (레어): 5 실드 + 반사 50%
🛡️ 무적 (에픽): 1턴 피해 무효

[마법 카드]
🔮 파이어볼 (커먼): 15 데미지
🔮 얼음창 (언커먼): 12 데미지 + 슬로우
🔮 번개 (레어): 25 데미지 + 연쇄
🔮 메테오 (에픽): 전체 40 데미지

[회복 카드]
💚 포션 (커먼): HP +10
💚 대형 포션 (언커먼): HP +25
💚 완전 회복 (레어): HP 전체 회복

[버프 카드]
⚡ 집중 (커먼): 다음 공격 +5
⚡ 광폭화 (언커먼): ATK +50% 3턴
⚡ 분신 (레어): 카드 복제 (1장)
```

### 2.3 카드 업그레이드

```
[업그레이드 규칙]
상점에서 골드로 업그레이드
업그레이드 시 "+" 붙음

[업그레이드 효과]
베기 → 베기+: 10 → 15 데미지
방어 → 방어+: 5 → 8 실드
파이어볼 → 파이어볼+: 15 → 22 데미지

[업그레이드 비용]
커먼: 25G
언커먼: 50G
레어: 100G
에픽: 200G
```

## 3. 전투 시스템 상세

### 3.1 전투 흐름

```
[턴 순서]
1. 덱에서 5장 드로우
2. 왼쪽부터 순서대로 자동 사용
3. 적 턴 (공격)
4. 2-3 반복
5. 승리/패배 판정

[덱 구성]
최대 덱 크기: 20장
최소 덱 크기: 5장
손패: 항상 5장
```

### 3.2 시너지 시스템

```
[시너지 조건]
같은 타입 카드 3장 → 보너스 발동

[시너지 효과]
공격 x3: 전체 공격력 +30%
방어 x3: 전체 방어력 +30%
마법 x3: 마법 데미지 +40%
회복 x3: 회복량 +50%
버프 x3: 버프 지속 +2턴

[고급 시너지]
같은 타입 5장: 보너스 효과 2배
3타입 혼합 (각 2장): 만능 +15%
```

### 3.3 데미지 계산

```
[데미지 공식]
최종 데미지 = 카드 데미지 × 시너지 보너스 × 버프 배수 - 적 방어력

[예시]
베기+ (15) × 공격 시너지 (1.3) × 광폭화 (1.5) - 적 방어 (5)
= 15 × 1.3 × 1.5 - 5 = 24.25 → 24 데미지
```

## 4. 던전 구조 상세

### 4.1 존 구성

```
[1런 = 3존]

존 1: 숲
├─ 스테이지 1-1: 러닝 500m + 전투 x2
├─ 스테이지 1-2: 러닝 600m + 전투 x3
├─ 스테이지 1-3: 러닝 700m + 전투 x2 + 미니보스
└─ 보상: 카드 선택 (3택 1)

존 2: 동굴
├─ 스테이지 2-1: 러닝 600m + 전투 x3
├─ 스테이지 2-2: 러닝 700m + 전투 x3
├─ 스테이지 2-3: 러닝 800m + 전투 x3 + 미니보스
└─ 보상: 카드 업그레이드 or 제거

존 3: 성
├─ 스테이지 3-1: 러닝 700m + 전투 x3
├─ 스테이지 3-2: 러닝 800m + 전투 x4
├─ 스테이지 3-3: 러닝 1000m + 최종보스
└─ 보상: 영구 보상
```

## 5. 상점 시스템

### 5.1 상점 구성

```
[상점 출현]
각 존 클리어 후 상점 등장
무작위 상품 4개 + 고정 상품 2개

[상품 종류]
카드 구매: 랜덤 카드 (50-200G)
카드 업그레이드: 보유 카드 강화 (25-200G)
카드 제거: 덱에서 카드 삭제 (75G)
체력 회복: HP +30 (25G)
유물 구매: 패시브 효과 (150-300G)
```

### 5.2 유물 시스템

```
[유물 예시]
🗡️ 날카로운 검: 공격 데미지 +3
🛡️ 강철 갑옷: 받는 데미지 -2
❤️ 생명의 반지: 최대 HP +20
⚡ 속도의 부츠: 첫 턴 2장 추가 드로우
🎲 행운의 주사위: 희귀 카드 확률 +10%
🔥 불의 문장: 화염 피해 +50%
💎 골드 마그넷: 골드 획득 +20%

[유물 슬롯]
최대 3개 장착 가능
```
