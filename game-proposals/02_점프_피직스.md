# 🦘 점프 피직스 (Jump Physics)

## 📋 기본 정보
- **장르**: 물리 기반 플랫포머 + 아이들러
- **플랫폼**: iOS / Android
- **타겟**: 전연령 캐주얼
- **예상 개발기간**: 3-4개월
- **팀 규모**: 1-3명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "터치 한 번으로 점프! 물리 법칙을 이용해 끝없이 높이 올라가는 원터치 점프 게임"

### 핵심 재미 요소
1. **원터치 점프** - 누르는 시간 = 점프 파워
2. **물리 인터랙션** - 벽 튕기기, 시소, 트램폴린
3. **무한 상승** - "얼마나 높이 갈 수 있나?"
4. **오프라인 수익** - 아이들러 요소로 자동 재화

---

## ⚙️ 핵심 메카닉

### 1. 점프 시스템
```
[조작]
- 터치: 점프 차징
- 릴리즈: 점프 실행
- 차징 시간 = 점프 높이 (최대 3초)

[물리 법칙]
- 중력: 9.8 (조절 가능)
- 벽 반발: 0.7
- 마찰: 0.3
```

### 2. 기믹 오브젝트
```
🟦 일반 플랫폼 - 기본 발판
🟨 트램폴린 - 2배 점프력
🟥 부서지는 발판 - 1회만 사용 가능
🟩 이동 플랫폼 - 좌우/상하 이동
⭕ 점프 링 - 통과 시 자동 슈퍼점프
🌀 포탈 - 다른 위치로 순간이동
💨 바람 - 좌우로 밀림
⚡ 부스터 - 3초간 무한 점프
```

### 3. 높이 시스템
```
[구간별 환경]
0-100m: 초원 (이지)
100-500m: 구름 (노멀)
500-1000m: 성층권 (하드)
1000-2000m: 우주 (익스트림)
2000m+: 은하계 (무한)

[환경 효과]
- 초원: 기본
- 구름: 바람 발생
- 성층권: 중력 감소
- 우주: 무중력 구간
- 은하계: 랜덤 이벤트
```

### 4. 캐릭터 성장
```
[업그레이드]
- 점프력: 기본 점프 높이 ↑
- 체력: 낙하 허용 횟수 ↑
- 마그넷: 코인 수집 범위 ↑
- 부스터 시간: 아이템 지속시간 ↑

[스킨]
- 기본 공
- 고양이 🐱
- 로봇 🤖
- 우주인 👨‍🚀
- (50종 이상)
```
