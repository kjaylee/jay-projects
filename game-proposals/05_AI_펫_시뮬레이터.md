# 🐾 AI 펫 시뮬레이터 (AI Pet Simulator)

## 📋 기본 정보
- **장르**: AI 기반 가상 펫 + 시뮬레이션
- **플랫폼**: iOS / Android
- **타겟**: 전연령 캐주얼 (특히 여성 10-35세)
- **예상 개발기간**: 4-5개월
- **팀 규모**: 2-3명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "AI가 만들어내는 유니크한 나만의 펫! 키우고, 대화하고, 함께 성장하는 차세대 다마고치"

### 핵심 재미 요소
1. **프로시저럴 펫** - AI 생성 유니크 외형
2. **대화형 교감** - 간단한 텍스트 대화
3. **성장 시뮬** - 먹이/놀이/훈련
4. **커뮤니티** - 펫 자랑 & 교배

---

## ⚙️ 핵심 메카닉

### 1. 펫 생성 시스템
```
[프로시저럴 생성]
시작 시 랜덤 시드 기반 펫 생성
- 종류: 고양이/강아지/토끼/판타지
- 색상: RGB 랜덤 조합
- 패턴: 20종 패턴 믹스
- 특징: 귀/꼬리/눈 변형

[DNA 시스템]
펫마다 고유 DNA 코드 (12자리)
예: A3F2-K9B1-X7C4

DNA로 펫 공유/교배 가능
```

### 2. 육성 시스템
```
[기본 스탯]
❤️ 체력: 0-100 (먹이로 회복)
😊 행복: 0-100 (놀이로 회복)
✨ 청결: 0-100 (목욕으로 회복)
⚡ 에너지: 0-100 (휴식으로 회복)

[성장 스탯]
💪 힘: 달리기 미니게임
🧠 지능: 퍼즐 미니게임
💕 친밀도: 상호작용 누적

[생애 주기]
알 (1일) → 아기 (3일) → 청소년 (7일) → 성체 (무한)
```

### 3. 상호작용
```
[기본 액션]
🍖 먹이주기: 음식 선택 → 드래그
🎾 놀아주기: 미니게임 (공던지기, 줄다리기)
🛁 목욕시키기: 문지르기 인터랙션
😴 재우기: 불 끄기 → 수면 모드
💬 대화하기: 텍스트 입력 → AI 반응

[반응 시스템]
펫 성격에 따라 다른 반응
- 활발: 신나서 뛰어다님
- 새침: 시큰둥하게 반응
- 겁쟁이: 숨거나 움츠림
```

### 4. AI 대화 시스템
```
[구현 방식]
로컬 경량 LLM 또는 서버 API

[대화 예시]
유저: "오늘 기분 어때?"
펫: "*꼬리를 흔들며* 주인이랑 같이 있어서 좋아! 오늘 간식 줄 거야?"

[성격 반영]
- 펫 성격에 따른 말투
- 스탯 상태 반영 (배고프면 불평)
- 학습된 단어 사용
```

### 5. 소셜 시스템
```
[펫 공유]
- DNA 코드로 공유
- 스크린샷 자동 생성
- SNS 공유 버튼

[교배]
DNA 2개 입력 → 새로운 펫 생성
부모 특성 믹스 + 돌연변이 확률

[랭킹]
- 가장 귀여운 펫 투표
- 미니게임 최고 기록
- 친밀도 랭킹
```
