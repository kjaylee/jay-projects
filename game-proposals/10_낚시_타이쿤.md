# 🎣 낚시 타이쿤 (Fishing Tycoon)

## 📋 기본 정보
- **장르**: 낚시 시뮬레이션 + 아이들러 타이쿤
- **플랫폼**: iOS / Android
- **타겟**: 전연령 힐링 캐주얼
- **예상 개발기간**: 4-5개월
- **팀 규모**: 2-4명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "물고기를 잡아 수족관을 키우고, 관광객을 모아 타이쿤이 되자! 힐링 낚시 + 경영 시뮬"

### 핵심 재미 요소
1. **힐링 낚시** - 물멍 + 낚시의 여유
2. **도감 수집** - 다양한 물고기
3. **수족관 경영** - 타이쿤 요소
4. **방치형 수익** - 오프라인 수입

---

## ⚙️ 핵심 메카닉

### 1. 낚시 시스템
```
[낚시 과정]
1. 캐스팅: 터치 & 홀드 → 릴리즈 (거리 결정)
2. 대기: 찌가 움직이면 터치 준비
3. 챔질: 타이밍 맞춰 터치
4. 릴링: 좌우 스와이프로 물고기 컨트롤

[판정]
- Perfect 챔질: 희귀 확률 ↑
- 릴링 성공: 물고기 획득
- 실패: 물고기 도망

[낚시터]
5개 지역 x 다른 어종
- 🏖️ 해변: 기본 어종
- 🌊 심해: 대형 어종
- 🏞️ 강: 민물 어종
- ❄️ 북극: 희귀 어종
- 🌴 열대: 열대어
```

### 2. 물고기 도감
```
[등급]
⬜ 커먼: 쉽게 잡힘
🟩 언커먼: 보통
🟦 레어: 특정 조건
🟪 에픽: 희귀
🟨 레전더리: 전설의 물고기

[물고기 예시]
🐟 고등어 (커먼): 어디서나
🐠 열대어 (언커먼): 열대 지역
🦈 상어 (레어): 심해, 밤
🐙 대왕오징어 (에픽): 심해, 폭풍우
🐉 용왕 (레전더리): 특수 이벤트

[도감 보상]
수집률에 따른 보너스
10% → 낚시대 ↑
50% → VIP 낚시터
100% → 전설의 낚시꾼 칭호
```

### 3. 수족관 시스템
```
[수족관 구성]
잡은 물고기를 수족관에 전시
물고기 종류/희귀도에 따른 입장료 수입

[시설]
🏛️ 수족관 홀: 물고기 전시
🍽️ 레스토랑: 방문객 체류시간 ↑
🎁 기념품샵: 추가 수익
🎢 체험존: 방문객 만족도 ↑

[업그레이드]
시설 레벨업 → 수용 인원/수익 ↑
```

### 4. 경영 시스템
```
[수익 구조]
- 입장료: 물고기 희귀도 비례
- 부대시설: 레스토랑, 샵 등
- 오프라인: 시간당 자동 수익

[방문객 시스템]
물고기 다양성 → 평점 ↑ → 방문객 ↑
특별 물고기 → 바이럴 이벤트
```

### 5. 장비 시스템
```
[낚싯대]
🎣 나무 낚싯대: 기본
🎣 카본 낚싯대: 캐스팅 ↑
🎣 티타늄 낚싯대: 희귀 확률 ↑
🎣 전설의 낚싯대: 올스탯 ↑

[릴]
⚙️ 기본 릴 → 고급 릴 → 전동 릴

[미끼]
🪱 지렁이 (무료)
🦐 새우 (커먼↑)
🐟 작은 물고기 (레어↑)
💎 황금 미끼 (에픽↑)
```
