# 🎲 럭키 다이스 마스터 (Lucky Dice Master)

## 📋 기본 정보
- **장르**: 주사위 로그라이크 덱빌더 (Yahtzee-like Roguelike)
- **플랫폼**: iOS / Android (세로모드)
- **타겟**: 캐주얼~미드코어 (20-45세)
- **예상 개발기간**: 3-5개월
- **팀 규모**: 1-3명
- **영감**: Balatro, Dicey Dungeons, Yahtzee

---

## 🎮 게임 컨셉

### 한 줄 요약
> "야찌 족보로 몬스터를 처치하고, 특수 주사위와 조커를 수집해 무한 콤보를 만들어라!"

### 핵심 아이디어
Balatro가 포커를 로그라이크로 재해석했듯이, **야찌(Yahtzee)** 를 로그라이크 덱빌더로 재해석합니다.

### 핵심 재미 요소
1. **족보의 쾌감** - 풀하우스, 라지 스트레이트 등 터지는 순간의 도파민
2. **빅 넘버** - 배수기가 중첩되어 숫자가 폭발적으로 커지는 쾌감
3. **빌드 구축** - 특수 주사위 + 조커 조합으로 나만의 빌드
4. **리스크-리워드** - 리롤 vs 확정, 도박 vs 안전 선택
5. **짧은 런** - 10-15분 완결, 반복 플레이 유도

## ⚙️ 핵심 메카닉

### 1. 기본 전투 시스템

```
[턴 구조]
1. 주사위 5개 굴리기 (기본)
2. 원하는 주사위 킵 → 나머지 리롤 (최대 2회)
3. 족보 선택 → 데미지 계산
4. 몬스터 HP 깎기
5. 몬스터 반격 (HP 감소)
6. 반복 → 몬스터 처치 또는 플레이어 사망
```

### 2. 족보 시스템

| 족보 | 조건 | 기본 데미지 | 배수 |
|------|------|------------|------|
| **원스** | 1이 1개 이상 | 주사위 합 | x1 |
| **투스** | 2가 1개 이상 | 주사위 합 | x1 |
| ... | ... | ... | ... |
| **트리플** | 같은 숫자 3개 | 주사위 합 | x2 |
| **포카드** | 같은 숫자 4개 | 주사위 합 | x4 |
| **풀하우스** | 트리플 + 페어 | 25 고정 | x3 |
| **스몰 스트레이트** | 연속 4개 (1-2-3-4 등) | 30 고정 | x4 |
| **라지 스트레이트** | 연속 5개 (1-2-3-4-5) | 40 고정 | x5 |
| **야찌** | 같은 숫자 5개 | 50 고정 | x10 |
| **찬스** | 아무거나 | 주사위 합 | x1 |

### 3. 특수 주사위 시스템

기본 주사위(1-6)를 **특수 주사위**로 교체하여 빌드를 구축합니다.

```
[주사위 등급]
⬜ 커먼 → 🟦 레어 → 🟪 에픽 → 🟨 레전더리

[특수 주사위 예시]
🔴 파이어 다이스: 6 나오면 주변 주사위 +1
🔵 아이스 다이스: 몬스터 공격 1턴 지연
🟢 럭키 다이스: 1이 6으로 변환 (50%)
⚡ 체인 다이스: 같은 숫자 나오면 추가 데미지
💀 커스드 다이스: 데미지 x2, 하지만 1 나오면 자해
🌈 와일드 다이스: 아무 숫자로 취급 가능
💎 프리즘 다이스: 족보 배수 +1
🎰 잭팟 다이스: 야찌 시 골드 x3
```

### 4. 조커 시스템 (Balatro 스타일)

런 중 획득하는 **조커 카드**로 족보를 강화합니다.

```
[조커 슬롯: 최대 5개]

[조커 예시]
🃏 더블러: 모든 배수 x2
🃏 스트레이트 러버: 스트레이트 데미지 +50
🃏 트리플 마스터: 트리플 이상에서 추가 리롤 +1
🃏 원스 킬러: 1이 3개 이상이면 즉사 확률 5%
🃏 야찌 헌터: 야찌 달성 시 다음 턴 자동 야찌
🃏 리롤 보너스: 리롤 안 하면 데미지 +20%
🃏 피의 계약: HP 50% 이하에서 데미지 x2
🃏 골드 다이거: 적 처치 시 골드 +50%
🃏 메탈 아머: 매 턴 시작 시 방어막 +5
🃏 페어 파워: 페어마다 배수 +0.5
```

### 5. 런 구조

```
[1런 = 3 앤테 x 3 블라인드]

앤테 1: 기본 몬스터 (목표 HP 100-300)
  ├─ 스몰 블라인드: 슬라임 (쉬움)
  ├─ 빅 블라인드: 고블린 (보통)
  └─ 보스 블라인드: 오크 대장 (특수 능력)

앤테 2: 중급 몬스터 (목표 HP 400-800)
  ├─ 스몰 블라인드
  ├─ 빅 블라인드
  └─ 보스 블라인드

앤테 3: 상급 몬스터 (목표 HP 1000-2000)
  ├─ 스몰 블라인드
  ├─ 빅 블라인드
  └─ 최종 보스

[블라인드 사이]
- 상점: 조커, 특수 주사위, 소모품 구매
- 이벤트: 랜덤 선택지 (리스크-리워드)
- 휴식: HP 회복 or 골드 보너스
```

### 6. 보스 메카닉

각 보스는 **특수 규칙**을 부여합니다.

| 보스 | 특수 능력 |
|------|----------|
| 🐲 **드래곤** | 화염 주사위만 유효 (다른 주사위 데미지 반감) |
| 👻 **유령왕** | 홀수 주사위 무시됨 |
| 🤖 **기계 골렘** | 스트레이트만 데미지 입음 |
| 🧙 **다크 메이지** | 매 턴 랜덤 주사위 1개 저주 (1로 고정) |
| 💀 **데스 나이트** | 야찌 아니면 반격 데미지 x2 |
| 🌙 **문 프리스트** | 6면 주사위가 7면으로 변경 |

## 📊 밸런스 설계

### 데미지 공식
```
최종 데미지 = (기본 데미지 × 족보 배수 × 조커 배수 × 주사위 보너스) 
              + 추가 고정 데미지
```

### 예시 계산
```
상황: 풀하우스 (6-6-6-3-3) + 더블러 조커 + 프리즘 다이스

기본 데미지: 25 (풀하우스 고정)
족보 배수: x3 (풀하우스)
조커 배수: x2 (더블러)
주사위 보너스: x1 + 1 = x2 (프리즘)

최종: 25 × 3 × 2 × 2 = 300 데미지
```

### 난이도 곡선

| 앤테 | 몬스터 HP | 플레이어 예상 DPS | 턴 수 |
|------|----------|-----------------|------|
| 1-1 | 100 | 30-50 | 2-4 |
| 1-3 (보스) | 300 | 60-100 | 3-5 |
| 2-1 | 400 | 100-150 | 3-4 |
| 2-3 (보스) | 800 | 200-300 | 3-4 |
| 3-1 | 1000 | 300-500 | 2-4 |
| 3-3 (최종) | 2000 | 500-1000 | 2-4 |

## ✅ 재미 분석

### 장점 (Pros)
- ✅ **검증된 코어 루프**: 야찌는 수십 년간 검증된 재미
- ✅ **즉각적 피드백**: 주사위 굴림의 시각/청각 만족감
- ✅ **전략적 깊이**: 리롤 타이밍, 빌드 선택
- ✅ **높은 리플레이 가치**: 매번 다른 빌드 가능
- ✅ **모바일 최적화**: 짧은 세션, 간단한 조작
- ✅ **트렌드 적합**: Balatro-like 장르의 성장세

### 단점 (Cons)
- ⚠️ **RNG 의존**: 운 요소가 클 수 있음 → 조커/주사위로 제어 가능하게
- ⚠️ **단조로움 가능성**: 반복 플레이 시 지루 → 다양한 조커/보스로 해결
- ⚠️ **경쟁작 존재**: Dicey Dungeons 등 → 모바일 특화로 차별화
- ⚠️ **밸런스 난이도**: 빌드 간 밸런스 조정 필요
