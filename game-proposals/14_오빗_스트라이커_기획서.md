# Orbit Striker (오빗 스트라이커) 기획서

## 1. 게임 개요

### 1.1 컨셉
**"중력을 무기로, 우주를 지켜라"**

행성 중심에서 탄환을 발사하여 궤도를 따라 도는 외계 침략자들을 격파하는 물리 기반 아케이드 슈터.

### 1.2 장르
- 아케이드 슈터 + 물리 퍼즐
- 캐주얼 액션

### 1.3 플랫폼
- 모바일 웹 (HTML5)
- 터치/클릭 조작

### 1.4 트렌드 분석
- 2026 캐주얼 게임 트렌드: 하이브리드 장르 (슈터 + 물리)
- Vampire Survivors 영향으로 웨이브 기반 게임 인기 지속
- 단순 조작 + 깊은 전략성 추구

---

## 2. 타겟 유저

### 2.1 주요 타겟
- **캐주얼 게이머**: 짧은 세션, 쉬운 조작
- **아케이드 팬**: 스코어 경쟁, 반복 플레이
- **연령대**: 15-35세

### 2.2 유저 니즈
- 5분 내 만족스러운 게임 경험
- 직관적 조작, 빠른 피드백
- 점수 갱신 욕구

---

## 3. 코어 루프

```
[조준] → [발사] → [궤도 계산/충돌] → [적 처치/보상] → [웨이브 진행]
                      ↓
               [파워업 획득] → [능력 강화]
```

### 3.1 게임 흐름
1. 행성 중심에 플레이어 캐논 위치
2. 터치/드래그로 발사 방향 조준
3. 탄환은 중력 영향 받아 곡선 궤도 이동
4. 적들은 행성 주위 궤도를 따라 회전
5. 적 처치 시 에너지 획득
6. 웨이브 클리어 → 파워업 선택
7. 무한 웨이브, 최고 점수 도전

---

## 4. 게임 메카닉

### 4.1 발사 시스템
- 드래그로 조준 (방향 + 힘 조절)
- 발사 후 탄환은 중력 곡선 따라 이동
- 탄환이 화면 밖으로 나가면 소멸

### 4.2 궤도 시스템
- 행성 중력: 탄환이 행성 방향으로 휘어짐
- 적들은 여러 궤도(내/중/외)에서 회전
- 궤도마다 다른 속도

### 4.3 적 타입
| 적 | HP | 특징 |
|---|---|---|
| Drifter | 1 | 느린 이동 |
| Orbiter | 2 | 일정 궤도 회전 |
| Swooper | 1 | 궤도 변경하며 접근 |
| Tank | 5 | 느리지만 강력 |
| Boss | 20 | 특수 패턴 |

### 4.4 파워업
- **멀티샷**: 탄환 3방향 발사
- **피어싱**: 관통 탄환
- **호밍**: 약한 유도 기능
- **빅샷**: 대형 탄환
- **속사**: 발사 쿨다운 감소
