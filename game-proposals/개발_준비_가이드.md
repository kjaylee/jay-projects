## 🏆 TOP 3 추천 게임

### 🥇 1위: 럭키 다이스 마스터 (총점: 25/25)

**선정 이유:**
1. ✅ **Balatro 트렌드:** 2024년 Balatro 대성공 → 모바일 클론 시장 아직 블루오션
2. ✅ **개발 기간:** 3개월 (가장 짧음)
3. ✅ **에셋 활용:** 보유 에셋만으로 100% 개발 가능
4. ✅ **경쟁도:** 야찌 기반 로그라이크 = 거의 없음
5. ✅ **수익 잠재력:** 덱빌더 장르 ARPDAU 높음

**핵심 강점:**
- 야찌(Yahtzee) = 검증된 50년 레거시
- 주사위 굴림의 물리적 만족감
- 빌드 구축의 전략적 깊이
- 모바일 세로모드 최적화

**리스크:**
- ⚠️ RNG 밸런싱 필요 → 조커/주사위로 제어 가능
- ⚠️ 반복 플레이 단조로움 → 다양한 보스 규칙으로 해결

### 🥈 2위: 스택 킹덤 (총점: 21/25)

**선정 이유:**
1. ✅ **심플한 코어:** 스태킹 메카닉 = 1일 프로토타입 가능
2. ✅ **시각적 보상:** 성이 커지는 것 = 즉각적 만족감
3. ✅ **에셋 활용:** KayKit Medieval Hexagon 완벽 활용
4. ✅ **바이럴 포텐셜:** 완성된 왕국 공유 → SNS 유입

**핵심 강점:**
- 원터치 조작 → 진입 장벽 낮음
- 건설 + 경쟁 = 두 가지 재미
- 시즌제 운영 가능

### 🥉 3위: 머지 몬스터 타워 (총점: 21/25)

**선정 이유:**
1. ✅ **하이브리드 장르:** 머지 + TD = 두 시장 커버
