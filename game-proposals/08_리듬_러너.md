# 🎵 리듬 러너 (Rhythm Runner)

## 📋 기본 정보
- **장르**: 리듬게임 + 오토러너
- **플랫폼**: iOS / Android
- **타겟**: 캐주얼~미드코어 (15-30세)
- **예상 개발기간**: 4-5개월
- **팀 규모**: 2-4명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "음악에 맞춰 달리고 점프! 비트에 맞추면 파워업되는 싱크로 러너"

### 핵심 재미 요소
1. **비트 싱크** - 음악과 액션의 일체감
2. **러너 심플함** - 누구나 즉시 플레이
3. **음악 다양성** - EDM, K-POP, 클래식
4. **유저 곡 지원** - 내 음악으로 플레이

---

## ⚙️ 핵심 메카닉

### 1. 러닝 시스템
```
[조작]
- 탭: 점프
- 홀드: 슬라이드
- 스와이프 좌/우: 레인 이동

[3레인 구조]
좌 | 중앙 | 우
장애물 배치는 비트에 맞춤
```

### 2. 리듬 시스템
```
[비트 노트]
음악 BPM에 맞춰 노트 생성
노트 타이밍에 맞춰 액션 → 판정

[판정]
- Perfect: 100점 + 파워게이지 ++
- Great: 80점 + 파워게이지 +
- Good: 50점
- Miss: 0점 + 콤보 리셋

[파워게이지]
Perfect/Great 누적 → 게이지 충전
풀 차지 시 피버타임 (10초)
- 무적 상태
- 점수 2배
- 비주얼 이펙트 강화
```

### 3. 스테이지 생성
```
[프리셋 곡]
50곡 프리셋 (다양한 장르)
전문 비트맵 제작

[유저 곡 (선택)]
유저 MP3 업로드
AI 비트 분석 → 자동 맵 생성
BPM, 피크, 드랍 감지

[난이도]
Easy: 기본 비트만
Normal: 서브 비트 추가
Hard: 복잡한 패턴
Expert: 풀 비트맵
```

### 4. 캐릭터 & 스킨
```
[캐릭터]
🏃 기본 러너
🎸 록스타
💃 댄서
🤖 로봇 DJ
👻 네온 고스트

[능력]
캐릭터별 패시브 스킬
- 록스타: Miss 1회 무효
- 댄서: 콤보 보너스 +20%
- 로봇: 피버 지속시간 +3초
```

### 5. 진행 시스템
```
[월드 투어]
5개 월드 x 10곡 = 50곡

[월드 테마]
🌆 시티 나이트 (EDM)
🎌 도쿄 스트릿 (J-POP)
🇰🇷 서울 바이브 (K-POP)
🎻 유럽 클래식 (오케스트라 리믹스)
🌈 네온 드림 (신스웨이브)

[언락 조건]
별 수집으로 다음 곡/월드 해금
```
