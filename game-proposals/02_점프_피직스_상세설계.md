## 1. 물리 시스템 상세

### 1.1 점프 메카닉

```
[점프 공식]
점프 높이 = 기본높이 × (차징시간/최대차징) × 점프력배수

기본높이: 50 유닛
최대차징: 3초
점프력배수: 1.0 (업그레이드로 증가)

[예시]
0.5초 차징: 50 × (0.5/3) = 8.3 유닛
1.5초 차징: 50 × (1.5/3) = 25 유닛
3.0초 차징: 50 × (3.0/3) = 50 유닛
```

### 1.2 물리 파라미터

```
[중력 설정]
기본 중력: 9.8 m/s²
성층권 (500m+): 6.0 m/s²
우주 (1000m+): 2.0 m/s²

[반발 계수]
일반 플랫폼: 0.0 (반발 없음)
벽: 0.7
트램폴린: 1.5
고무 발판: 1.2

[마찰 계수]
일반: 0.3
얼음: 0.05
모래: 0.6
```

### 1.3 캐릭터 물리

```
[이동 속도]
기본 낙하 속도: 최대 20 m/s
좌우 이동: 5 m/s (공중)
질량: 1.0 (고정)

[충돌 판정]
원형 콜라이더: 반경 0.5 유닛
플랫폼 판정: 발 기준 (상단만 충돌)
```

## 2. 기믹 오브젝트 상세

### 2.1 플랫폼 종류

| 오브젝트 | 효과 | 시각 피드백 |
|----------|------|------------|
| 🟦 일반 | 기본 발판 | 없음 |
| 🟨 트램폴린 | 점프력 x2 | 스프링 애니메이션 |
| 🟥 부서지는 | 1회 착지 후 파괴 | 흔들림 → 파괴 |
| 🟩 이동 | 좌우 이동 (2초 주기) | 화살표 표시 |
| ⭕ 점프링 | 통과 시 자동 점프 | 빛남 이펙트 |
| 🌀 포탈 | 순간이동 | 파티클 빨려들기 |
| 💨 바람 | 방향으로 밀림 | 바람 파티클 |
| ⚡ 부스터 | 3초간 무한 점프 | 아우라 이펙트 |
| ☁️ 구름 | 슬로우 폴링 | 뭉게뭉게 애니메이션 |
| 🔥 용암 | 즉사 | 불꽃 파티클 |

### 2.2 오브젝트 배치 확률

```
[높이별 배치]
0-100m: 일반 80%, 트램폴린 15%, 점프링 5%
100-500m: 일반 60%, 이동 20%, 부서지는 15%, 기타 5%
500-1000m: 일반 50%, 부서지는 25%, 바람 15%, 위험 10%
1000m+: 랜덤 혼합, 난이도 최고

[플랫폼 간격]
0-100m: 30-50 유닛
100-500m: 40-70 유닛
500m+: 50-100 유닛
```

## 3. 높이 구간 시스템

### 3.1 환경 변화

| 구간 | 배경 | 특수 효과 | BGM |
|------|------|----------|-----|
| 0-100m 초원 | 푸른 하늘, 구름 | 없음 | 밝은 캐주얼 |
| 100-500m 구름 | 구름 속 | 바람 발생 | 몽환적 |
| 500-1000m 성층권 | 어두운 하늘 | 중력 감소 | 긴장감 |
| 1000-2000m 우주 | 별, 행성 | 무중력 구간 | 스페이스 앰비언트 |
| 2000m+ 은하계 | 은하수 | 랜덤 이벤트 | 에픽 |

### 3.2 특수 이벤트

```
[은하계 랜덤 이벤트]
- 유성우: 피해야 하는 장애물
- 블랙홀: 빨려들면 랜덤 위치로 이동
- 별똥별: 터치 시 보너스 점프
- UFO: 따라가면 보너스 구간
```

## 6. 아이들러 시스템

### 6.1 오프라인 수익

```
[계산 공식]
오프라인 골드 = 최고기록(m) × 시간(h) × 0.5

예시:
최고기록 500m, 8시간 접속 안함
= 500 × 8 × 0.5 = 2000 골드

[상한]
최대 24시간 누적
VIP: 48시간 누적
```

### 6.2 퀵 수익

```
[광고 보상]
- 2시간 오프라인 수익 즉시 획득
- 하루 5회 제한
```
