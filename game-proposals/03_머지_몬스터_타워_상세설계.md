## 1. 머지 시스템 상세

### 1.1 머지 규칙

```
[기본 규칙]
같은 레벨 몬스터 2개 드래그 → 레벨+1 몬스터 1개

[레벨 티어]
Lv.1-10: 노멀 (회색 테두리)
Lv.11-20: 레어 (파란 테두리)
Lv.21-30: 에픽 (보라 테두리)
Lv.31-40: 레전더리 (금색 테두리)
Lv.41-50: 미식 (무지개 테두리)
```

### 1.2 변이 시스템

```
[변이 트리거]
Lv.10 → Lv.11 합성 시: 10% 확률로 특수 변이
Lv.20 → Lv.21 합성 시: 15% 확률로 특수 변이
Lv.30 → Lv.31 합성 시: 20% 확률로 특수 변이

[변이체 예시]
파이어 고블린 Lv.10 → 인페르노 로드 Lv.11 (변이)
아이스 임프 Lv.20 → 프로스트 타이탄 Lv.21 (변이)

[변이 효과]
변이체는 기본 스탯 x1.5 + 특수 스킬
```

## 2. 몬스터 클래스 상세

### 2.1 클래스 스탯 보정

| 클래스 | HP | ATK | 공속 | 사거리 | 특수 |
|--------|-----|-----|------|-------|------|
| ⚔️ 워리어 | +30% | +10% | 기본 | 근접 | 타겟 고정 |
| 🏹 아처 | -10% | +20% | 빠름 | 원거리 | 관통 |
| 🔮 메이지 | -20% | +30% | 느림 | 원거리 | 범위 공격 |
| 🛡️ 탱커 | +100% | -30% | 느림 | 근접 | 도발 |
| 💚 힐러 | -30% | -50% | 기본 | 원거리 | 아군 힐 |
| ⚡ 어쌔신 | -20% | +40% | 최고속 | 근접 | 치명타 50% |

### 2.2 클래스별 스킬

```
[워리어 스킬]
Lv.1-10: 강타 (단일 150% 데미지)
Lv.11-20: 회전베기 (주변 3체 공격)
Lv.21+: 처형 (HP 20% 이하 즉사)

[아처 스킬]
Lv.1-10: 관통 화살 (2체 관통)
Lv.11-20: 화살비 (범위 70% x 5회)
Lv.21+: 저격 (최후방 적 300% 데미지)

[메이지 스킬]
Lv.1-10: 파이어볼 (범위 120%)
Lv.11-20: 메테오 (범위 200% + 스턴)
Lv.21+: 절대영도 (전체 슬로우 50%)

[탱커 스킬]
Lv.1-10: 도발 (3초간 타겟 집중)
Lv.11-20: 철벽 (5초 피해 70% 감소)
Lv.21+: 반사 (받은 피해 30% 반사)

[힐러 스킬]
Lv.1-10: 치유 (아군 1체 HP 20% 회복)
Lv.11-20: 치유의 서클 (전체 HP 10% 회복)
Lv.21+: 부활 (사망한 아군 1체 30% HP 부활)

[어쌔신 스킬]
Lv.1-10: 급소 공격 (치명타 확정)
Lv.11-20: 그림자 이동 (후방 침투)
Lv.21+: 연속 암살 (킬 시 즉시 행동)
```

## 4. 웨이브 디펜스 상세

### 4.1 웨이브 구성

```
[1스테이지 = 10웨이브]

웨이브 1: 일반몹 x5 (HP 100)
웨이브 2: 일반몹 x8 (HP 120)
웨이브 3: 일반몹 x10 + 빠른몹 x3
웨이브 4: 정예몹 x2 + 일반몹 x5
웨이브 5: 중간보스 (HP 1000)
웨이브 6: 일반몹 러시 x15
웨이브 7: 정예몹 x4 + 힐러몹 x1
웨이브 8: 비행몹 x5 + 일반몹 x10
웨이브 9: 정예몹 러시 x8
웨이브 10: 보스 (HP 5000) + 미니언 x5
```

### 4.2 적 타입

| 적 타입 | 특징 | 대응 |
|---------|------|------|
| 일반몹 | 기본 | 모든 공격 |
| 빠른몹 | 이속 x2 | 슬로우, 전방 배치 |
| 정예몹 | 스탯 x2 | 집중 공격 |
| 힐러몹 | 주변 힐 | 우선 처치 |
| 비행몹 | 지상 무시 | 아처, 메이지만 |
| 탱크몹 | HP x5, 느림 | 지속 딜 |
| 보스 | 특수 스킬 | 전략 필요 |

### 4.3 스테이지 스케일링

```
[스케일링 공식]
적 HP = 기본HP × (1.05 ^ 스테이지)
적 ATK = 기본ATK × (1.03 ^ 스테이지)

예시:
스테이지 1: HP 100, ATK 10
스테이지 10: HP 163, ATK 13
스테이지 50: HP 1147, ATK 44
스테이지 100: HP 13150, ATK 196
```

## 5. 아이들러 시스템

### 5.1 오프라인 전투

```
[계산 방식]
마지막 클리어 스테이지 자동 반복
시간당 예상 클리어 횟수 계산

오프라인 골드 = 스테이지보상 × 클리어횟수 × 0.5
오프라인 경험치 = 스테이지EXP × 클리어횟수 × 0.3

[상한]
최대 24시간 누적
프리미엄: 48시간 누적
```

### 5.2 퀵 배틀

```
[기능]
광고 시청 → 2시간 오프라인 보상 즉시 획득
하루 10회 제한
VIP: 무제한
```
