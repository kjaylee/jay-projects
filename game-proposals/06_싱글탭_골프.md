# ⛳ 싱글탭 골프 (Single Tap Golf)

## 📋 기본 정보
- **장르**: 하이퍼캐주얼 골프 + 퍼즐
- **플랫폼**: iOS / Android
- **타겟**: 전연령 울트라캐주얼
- **예상 개발기간**: 2-3개월
- **팀 규모**: 1-2명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "탭 한 번으로 홀인원! 타이밍과 각도만으로 승부하는 미니멀 골프"

### 핵심 재미 요소
1. **원터치 조작** - 탭만으로 플레이
2. **물리 재미** - 공이 굴러가는 만족감
3. **퍼즐 요소** - 장애물 활용 전략
4. **짧은 세션** - 1홀 10초

---

## ⚙️ 핵심 메카닉

### 1. 조작 시스템
```
[원터치 메카닉]
- 공이 자동으로 조준선 회전
- 터치하면 그 방향으로 발사
- 터치 유지 시간 = 파워

[조작 옵션]
- Easy: 조준선 느리게 회전
- Normal: 보통 속도
- Hard: 빠른 회전 + 파워 게이지
```

### 2. 코스 시스템
```
[코스 구성]
5개 월드 x 20홀 = 100홀+

[월드 테마]
🌳 초원: 기본 (튜토리얼)
🏜️ 사막: 모래 트랩 (슬로우)
❄️ 눈밭: 얼음 (미끄럼)
🌋 화산: 용암 (즉사)
🌙 우주: 무중력 (특수 물리)

[홀 구조]
- 티박스 (시작)
- 페어웨이 (일반 지형)
- 장애물 (벙커, 물, 벽)
- 기믹 (포탈, 경사, 점프대)
- 홀 (목표)
```

### 3. 장애물 & 기믹
```
[장애물]
🟫 벙커: 파워 50% 감소
💧 워터: 1타 페널티 리셋
🧱 벽: 반사
🌀 모래폭풍: 방향 랜덤 변경

[기믹]
🔵 포탈: 순간이동
⬆️ 점프대: 공중 점프
↗️ 경사면: 방향 전환
💨 바람: 지속적 힘
⚡ 부스터: 2배 파워
🧲 마그넷: 홀 흡입력
```

### 4. 점수 시스템
```
[스코어]
- 홀인원: ⭐⭐⭐ + 보너스
- 버디 (-1): ⭐⭐
- 파 (0): ⭐
- 보기 (+1): 클리어
- +2 이상: 실패

[별 보상]
별로 다음 월드 언락
추가 골프공 스킨 구매
```

### 5. 컬렉션
```
[골프공]
🔵 기본공
🔴 파워공 (+10% 파워)
🟢 컨트롤공 (+조준선 느림)
⭐ 럭키공 (+홀 마그넷)
🌈 레인보우공 (이펙트)

[플레이어 스킨]
다양한 골퍼 캐릭터
(영향 없음, 순수 꾸미기)
```
