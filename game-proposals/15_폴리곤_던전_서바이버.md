## 🎮 게임 컨셉

### 핵심 재미 요소
1. **뱀서라이크 액션** — 쏟아지는 적들, 자동 공격, 레벨업 선택
2. **POLYGON 아트** — 깔끔한 로우폴리 비주얼
3. **로그라이크 성장** — 매 런마다 다른 빌드
4. **던전 탐험** — 다양한 던전 테마

### 차별점
- **뱀파이어 서바이버** + **던전 테마** + **POLYGON 스타일**
- 보유 에셋 100% 활용으로 빠른 개발
- Feel 에셋으로 주스 가득한 타격감

## ⚙️ 핵심 메카닉

### 1. 전투 시스템
```
[자동 공격]
- 가장 가까운 적 자동 타겟팅
- 무기별 공격 패턴
- Feel 에셋으로 타격 피드백 (스크린쉐이크, 햅틱)

[무기 종류]
- 검: 근접 휘두르기
- 활: 원거리 자동 발사
- 마법: 범위 공격
- 소환: 스켈레톤 소환
```

### 2. 레벨업 시스템
```
[경험치]
- 적 처치 시 젬 드랍
- 젬 수집 → 경험치
- 레벨업 시 3개 선택지

[업그레이드 종류]
- 새 무기 획득
- 무기 강화 (데미지, 속도, 범위)
- 패시브 스킬 (이속, 체력, 회복)
```

### 3. 던전 구조
```
[스테이지 구성]
- 스테이지 1: 지하 감옥 (5분)
- 스테이지 2: 묘지 (7분)
- 스테이지 3: 보스 던전 (3분)

[보스]
- 각 스테이지 종료 시 보스 등장
- 보스 패턴 회피 + 공격
```

### 4. 메타 진행
```
[영구 업그레이드]
- 골드로 구매
- 기본 스탯 증가
- 새 캐릭터 해금
- 새 무기 해금
```
