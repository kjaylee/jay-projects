# 🏰 머지 몬스터 타워 (Merge Monster Tower)

## 📋 기본 정보
- **장르**: 머지 + 타워디펜스 + 아이들러
- **플랫폼**: iOS / Android
- **타겟**: 캐주얼~미드코어 (18-45세)
- **예상 개발기간**: 4-5개월
- **팀 규모**: 2-4명

---

## 🎮 게임 컨셉

### 한 줄 요약
> "몬스터를 합쳐서 강해지고, 타워를 쌓아 적을 막아라! 머지 + TD의 완벽한 조합"

### 핵심 재미 요소
1. **머지 중독** - 합치면 강해지는 도파민
2. **전략적 배치** - 어디에 뭘 놓을까
3. **방치형 성장** - 오프라인에서도 전투
4. **보스 레이드** - 길드 협동 콘텐츠

---

## ⚙️ 핵심 메카닉

### 1. 머지 시스템
```
[머지 규칙]
동일 몬스터 2개 드래그 → 상위 레벨 1개

[레벨 티어]
Lv.1-10: 노멀 (자동 생성)
Lv.11-20: 레어 (합성으로만)
Lv.21-30: 에픽 (확률적 변이)
Lv.31-40: 레전더리 (특수 조합)
Lv.41-50: 미식 (시즌 한정)

[변이 시스템]
Lv.20 → Lv.21 합성 시 10% 확률로 특수 변이체
예: 파이어 고블린 → 인페르노 로드
```

### 2. 몬스터 클래스
```
⚔️ 워리어 - 근접, 높은 체력
🏹 아처 - 원거리, 빠른 공속
🔮 메이지 - 범위 공격, 디버프
🛡️ 탱커 - 도발, 초고체력
💚 힐러 - 아군 회복
⚡ 어쌔신 - 치명타, 순간 딜
```

### 3. 타워 시스템
```
[그리드 구조]
5x8 배치 그리드 (40칸)

[타워 종류]
- 메인 타워: 몬스터 배치 가능
- 서포트 타워: 버프 제공 (공속, 공격력 등)
- 트랩 타워: 적 슬로우/데미지

[업그레이드]
타워 레벨업 → 배치 가능 몬스터 수 ↑
```

### 4. 웨이브 디펜스
```
[1스테이지 = 10웨이브]
- 웨이브 1-3: 일반 몹
- 웨이브 4-6: 정예 몹
- 웨이브 7-9: 다수 러시
- 웨이브 10: 보스

[난이도 스케일링]
스테이지 클리어 → 다음 스테이지 적 체력/공격력 +5%
무한 스케일링 (푸쉬 시스템)
```

### 5. 아이들러 요소
```
[오프라인 전투]
- 마지막 클리어 스테이지 자동 반복
- 시간당 골드/경험치 획득
- 최대 24시간 누적

[퀵 배틀]
- 광고 시청 → 2시간 오프라인 보상 즉시 획득
```
