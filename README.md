## 🎲 개발 예정: 럭키 다이스 마스터

### 컨셉
> "야찌 족보로 몬스터를 처치하고, 특수 주사위와 조커를 수집해 무한 콤보를 만들어라!"

### 핵심 특징
- 🎯 **Balatro 스타일** 로그라이크 덱빌더
- 🎲 **야찌(Yahtzee)** 기반 족보 시스템
- 📱 **모바일 세로모드** 최적화
- ⏱️ **10-15분** 1런 완결

### MVP 범위
- 주사위 5개 + 족보 10종
- 특수 주사위 5종 + 조커 10종
- 3 앤테 x 3 블라인드 (9 스테이지)
- 보스 3종 (특수 규칙)
- 상점 시스템

### 개발 일정
- **Sprint 1 (Week 1-2):** 코어 루프 - 주사위 굴리기, 족보 판정, 전투
- **Sprint 2 (Week 3-4):** 런 구조 - 조커, 상점, 보스
- **Sprint 3-6 (Week 5-12):** 폴리싱 & 수익화
