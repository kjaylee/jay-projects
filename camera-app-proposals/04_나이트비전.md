# 🌙 나이트비전 (NightVision)

## 📋 기본 정보
- **카테고리**: 저조도/야경 전문 카메라
- **플랫폼**: iOS / Android
- **타겟**: 야경 촬영, 천체 사진, 클럽/파티 촬영
- **예상 개발기간**: 4-5개월
- **팀 규모**: 2-4명

## 📱 앱 컨셉

### 한 줄 요약
> "어두워도 선명하게! AI 저조도 향상으로 밤을 낮처럼 찍는 카메라"

### 핵심 차별점
1. **AI 저조도 향상** — 실시간 밝기 증폭
2. **노이즈 제거** — AI 기반 깨끗한 이미지
3. **천체 모드** — 별, 달, 은하수 촬영
4. **장노출 합성** — 손떨림 보정 + 합성

## ⚙️ 핵심 기능

### 1. AI 나이트 모드
```
[실시간 향상]
- 라이브 프리뷰에서 밝기 증폭
- AI 노이즈 제거
- 디테일 복원

[자동 감지]
조도 측정 → 최적 설정 자동 적용

[수동 조절]
밝기 증폭: 1x ―――● 10x
노이즈 제거: 약 ―――● 강
선명도: 자연 ―――● 샤프
```

### 2. 장노출 합성
```
[작동 방식]
여러 장 연속 촬영 → AI 정렬 → 합성

[모드]
🚗 라이트 트레일 — 차량 불빛 궤적
💧 물 흐름 — 실크같은 물
⭐ 별 궤적 — 스타 트레일
🏙️ 시티스케이프 — HDR 야경

[손떨림 보정]
- AI 프레임 정렬
- 삼각대 없이 장노출 효과
```

### 3. 천체 모드
```
[별 촬영]
- 별 감지 & 강조
- 별자리 오버레이
- 최적 노출 자동 계산

[달 촬영]
- 달 노출 최적화
- 크레이터 디테일
- 달 위상 정보

[은하수]
- 은하수 방향 AR 가이드
- 장노출 스태킹
- 빛공해 필터

[유성우]
- 연속 촬영 모드
- 유성 자동 감지
- 합성 이미지 생성
```

### 4. 파티/클럽 모드
```
[특수 조명 대응]
- 네온/LED 색 보정
- 깜빡임 방지
- 피부톤 복원

[모션 캡처]
- 빠른 셔터 스피드
- 동작 프리즈
- 블러 효과 (의도적)
```

### 5. RAW & 프로 컨트롤
```
[RAW 지원]
- DNG 포맷 저장
- 16bit 색심도
- 후보정 여유

[수동 컨트롤]
- ISO: 50-12800
- 셔터: 1/8000 - 30초
- 초점: 무한대~매크로
- 화이트밸런스: K 단위
```
