# 📄 스캔잇 (ScanIt)

## 📋 기본 정보
- **카테고리**: AI 문서 스캐너
- **플랫폼**: iOS / Android
- **타겟**: 학생, 직장인, 비즈니스
- **예상 개발기간**: 3-4개월
- **팀 규모**: 2-3명

---

## 📱 앱 컨셉

### 한 줄 요약
> "찍으면 바로 PDF! AI가 자동으로 보정하고 텍스트까지 추출하는 스마트 스캐너"

### 핵심 차별점
1. **자동 문서 감지** — 기울어져도 자동 보정
2. **AI OCR** — 손글씨도 인식
3. **다국어 번역** — 실시간 번역
4. **스마트 정리** — 자동 분류 & 검색

## ⚙️ 핵심 기능

### 1. 스마트 스캔
```
[자동 감지]
- 문서 가장자리 자동 인식
- 기울기 자동 보정
- 원근 왜곡 교정

[촬영 모드]
📄 단일 페이지
📚 연속 스캔 (다중 페이지)
🪪 ID카드 (앞뒤)
📇 명함
🧾 영수증
📋 화이트보드
```

### 2. AI 이미지 처리
```
[자동 보정]
- 조명 균일화
- 그림자 제거
- 노이즈 제거
- 선명도 향상

[필터]
⬜ 원본
📝 문서 (흑백 최적화)
🎨 컬러 문서
📰 잡지/사진
✏️ 화이트보드
```

### 3. OCR (텍스트 인식)
```
[인식 언어]
🇰🇷 한국어
🇺🇸 영어
🇯🇵 일본어
🇨🇳 중국어
+ 50개 언어

[인식 대상]
- 인쇄 텍스트
- 손글씨 (AI)
- 표/테이블
- 수식

[출력 형식]
- 텍스트 복사
- Word (.docx)
- Excel (.xlsx) - 표
- 검색 가능 PDF
```

### 4. 실시간 번역
```
[번역 기능]
스캔 → OCR → 번역 → 오버레이 표시

[지원 언어]
100+ 언어 쌍

[활용]
- 해외 문서 즉시 이해
- 여행 중 메뉴/안내판
- 외국어 학습
```

### 5. 문서 관리
```
[자동 분류]
AI가 문서 유형 인식
📂 계약서
📂 영수증
📂 명함
📂 노트
📂 기타

[스마트 검색]
OCR 텍스트 기반 전문 검색
"계약서" 검색 → 모든 계약서 문서

[태그 & 폴더]
- 커스텀 태그
- 중첩 폴더
- 즐겨찾기
```
