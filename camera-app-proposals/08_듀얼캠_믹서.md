## ⚙️ 핵심 기능

### 1. 듀얼 녹화
```
[카메라 소스]
📷 전면 카메라
📷 후면 카메라 (광각/망원 전환)
📱 외부 카메라 (연결 시)

[동시 녹화]
- 두 카메라 싱크 녹화
- 개별 파일 저장 옵션
- 합성 파일 저장
```

### 2. 레이아웃 모드
```
[PIP (Picture in Picture)]
┌─────────────┐
│             │
│    메인     │ ┌───┐
│             │ │서브│
│             │ └───┘
└─────────────┘
- 서브 위치: 4코너
- 서브 크기: 조절 가능
- 서브 모양: 원형/사각/라운드

[분할 화면]
┌──────┬──────┐
│      │      │
│ 전면 │ 후면 │
│      │      │
└──────┴──────┘
- 좌우 / 상하 분할
- 비율: 50:50, 70:30, 30:70

[블렌드]
- 오버레이 합성
- 투명도 조절
- 크로마키 (그린스크린)
```

### 3. 실시간 스위칭
```
[라이브 전환]
촬영 중 탭으로 메인/서브 전환

[전환 효과]
- 컷 (즉시)
- 디졸브 (페이드)
- 와이프 (방향 선택)
- 줌 전환

[자동 전환]
- 얼굴 감지 → 말하는 사람 자동 전환
- 타이머 전환 (N초마다)
```

### 4. 숏폼 최적화
```
[포맷]
- 9:16 (TikTok, Reels, Shorts)
- 16:9 (YouTube)
- 1:1 (Instagram)
- 4:5 (Instagram 피드)

[프리셋 레이아웃]
🎬 리액션 — 상단 콘텐츠 + 하단 얼굴
📱 브이로그 — 메인 풍경 + PIP 얼굴
🎤 인터뷰 — 좌우 분할
🎮 게임 — 화면녹화 + 페이스캠
```

### 5. 오디오 컨트롤
```
[마이크 소스]
🎙️ 전면 마이크
🎙️ 후면 마이크
🎧 외부 마이크
🎵 배경음악

[믹싱]
- 각 소스 볼륨 개별 조절
- 노이즈 캔슬링
- 음성 강조
```

### 6. 라이브 스트리밍
```
[지원 플랫폼]
- YouTube Live
- Instagram Live
- TikTok Live
- Twitch
- RTMP 커스텀

[스트리밍 옵션]
- 화질 선택 (720p/1080p)
- 비트레이트 조절
- 채팅 오버레이
```
