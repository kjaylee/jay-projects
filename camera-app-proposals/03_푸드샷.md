# 🍔 푸드샷 (FoodShot)

## 📋 기본 정보
- **카테고리**: 음식 전문 카메라
- **플랫폼**: iOS / Android
- **타겟**: 20-40세, 맛집 탐방/인스타 푸드스타그래머
- **예상 개발기간**: 3-4개월
- **팀 규모**: 2-3명

---

## 📱 앱 컨셉

### 한 줄 요약
> "음식 사진 전용 카메라! 찍으면 자동으로 맛있어 보이는 필터와 구도 가이드"

### 핵심 차별점
1. **음식 특화 AI** — 음식만 인식해서 최적화
2. **구도 가이드** — 플랫레이/45도/클로즈업 안내
3. **맛있어 보이는 필터** — 음식 전용 프리셋
4. **메뉴 인식** — 음식 이름 자동 태깅

---

## ⚙️ 핵심 기능

### 1. 음식 AI 보정
```
[자동 인식]
음식 촬영 → AI가 음식 영역 감지

[최적화 항목]
🔥 따뜻함 — 음식 온도감 표현
💧 수분감 — 신선함 강조
✨ 광택 — 윤기 추가
🎨 색감 — 채도 자연스럽게 UP
📍 포커스 — 음식에 초점, 배경 블러
```

### 2. 구도 가이드
```
[촬영 모드]
📐 플랫레이 (Top-down)
   - 수평 가이드 표시
   - 그리드 오버레이
   
📷 45도 앵글
   - 최적 각도 안내
   - 접시 정렬 가이드
   
🔍 클로즈업
   - 포커스 포인트 추천
   - 배경 블러 프리뷰

🍽️ 테이블 세팅
   - 여러 접시 배치 가이드
   - 대칭/비대칭 추천
```

### 3. 푸드 필터
```
[카테고리별 필터]
🍕 패스트푸드 — 비비드, 따뜻함
🍣 일식 — 깔끔, 차가운 톤
🥘 한식 — 따뜻함, 풍성함
🥗 샐러드 — 선명, 신선함
🍰 디저트 — 파스텔, 달콤함
☕ 카페 — 무드, 빈티지
🍺 주류 — 어두움, 분위기

[강도 조절]
살짝 ←――●――→ 맛있게!
```

### 4. 메뉴 인식 & 태깅
```
[AI 음식 인식]
사진 촬영 → "김치찌개" 자동 인식

[자동 태깅]
#김치찌개 #한식 #집밥 #맛스타그램

[맛집 정보 연동]
- 현재 위치 기반 식당 추천
- 네이버/카카오맵 연동
- 가격/별점 표시
```

### 5. 소셜 기능
```
[피드]
- 다른 유저 음식 사진
- 인기 맛집 트렌딩
- 팔로우/좋아요

[맛집 지도]
- 내가 찍은 음식 지도에 표시
- 다른 유저 사진 보기
- 근처 인기 맛집

[챌린지]
- 주간 음식 테마 챌린지
- 베스트 푸드샷 선정
```
