# 🔒 워터마크 캠 (Watermark Cam)

## 📋 기본 정보
- **카테고리**: 저작권 보호 카메라
- **플랫폼**: iOS / Android
- **타겟**: 크리에이터, 부동산, 공사현장, 비즈니스
- **예상 개발기간**: 2-3개월
- **팀 규모**: 1-3명

---

## 📱 앱 컨셉

### 한 줄 요약
> "찍는 순간 워터마크 & 타임스탬프! 저작권 보호와 증빙용 사진을 위한 카메라"

### 핵심 차별점
1. **자동 워터마크** — 촬영 시 즉시 적용
2. **위변조 방지** — 타임스탬프 + GPS
3. **템플릿 시스템** — 용도별 워터마크
4. **배치 처리** — 기존 사진에도 적용

---

## ⚙️ 핵심 기능

### 1. 워터마크 시스템
```
[워터마크 타입]
📝 텍스트 — 이름, 회사명, 저작권
🖼️ 이미지 — 로고, 서명
📅 타임스탬프 — 날짜/시간 자동
📍 위치정보 — GPS 좌표/주소
🔢 일련번호 — 자동 넘버링

[커스터마이징]
- 폰트: 50종+
- 색상: 자유 선택
- 투명도: 0-100%
- 크기: 자유 조절
- 위치: 9포인트 + 드래그
```

### 2. 타임스탬프 옵션
```
[표시 형식]
- 2026.01.27 12:35:00
- 2026/01/27 오후 12:35
- Jan 27, 2026 12:35 PM
- 커스텀 포맷

[추가 정보]
- 요일 표시
- 타임존
- 음력 날짜
- 촬영 설정 (ISO, SS)
```

### 3. GPS & 위치
```
[위치 표시]
📍 좌표: 37.5665° N, 126.9780° E
🏠 주소: 서울특별시 중구 세종대로 110
🗺️ 지도 미니맵 삽입

[정확도 표시]
- GPS 정확도 ±5m
- 고도 표시
- 방향(컴퍼스)
```

### 4. 용도별 템플릿
```
[프리셋 템플릿]
📸 크리에이터 — 로고 + © + 이름
🏠 부동산 — 날짜 + 주소 + 면적
🏗️ 공사현장 — 날짜 + 위치 + 공정
📦 배송/물류 — 시간 + 바코드 영역
⚖️ 법적 증빙 — 풀 메타데이터
🎨 아티스트 — 서명 + 연도

[커스텀 템플릿]
- 나만의 템플릿 저장
- 최대 20개 저장
- 빠른 전환
```

### 5. 배치 처리
```
[기존 사진 적용]
갤러리에서 다중 선택
→ 템플릿 선택
→ 일괄 적용
→ 내보내기

[옵션]
- 원본 유지/덮어쓰기
- 폴더 지정
- 파일명 규칙
```

### 6. 보안 기능
```
[위변조 방지]
- C2PA 메타데이터 (선택)
- 해시값 기록
- 블록체인 타임스탬프 (프리미엄)

[내보내기 옵션]
- 원본 + 워터마크 분리 저장
- 메타데이터 포함/제외
- 품질 선택
```
