## 📱 앱 컨셉

### 한 줄 요약
> "자연스럽게 예뻐지는 AI 셀카! 성형 아닌 자연스러운 보정의 정석"

### 핵심 차별점
1. **자연스러운 뷰티** — 과하지 않은 AI 보정
2. **얼굴형 분석** — 개인 맞춤 보정
3. **조명 시뮬레이션** — 가상 스튜디오 조명
4. **포즈 가이드** — 베스트 앵글 안내

---

## ⚙️ 핵심 기능

### 1. AI 뷰티 보정
```
[자동 분석]
얼굴 68포인트 감지 → 개인 맞춤 보정

[보정 항목]
👁️ 눈 — 크기, 밝기, 속눈썹
👃 코 — 높이, 너비 (미세 조정)
👄 입술 — 색상, 볼륨
🦷 치아 — 화이트닝
✨ 피부 — 잡티, 모공, 톤
📐 얼굴형 — V라인 (자연스럽게)

[강도 조절]
내추럴 ←――●――→ 글램
(AI 추천 기본값)
```

### 2. 가상 메이크업
```
[실시간 AR 메이크업]
💄 립스틱 — 색상/질감 선택
👁️ 아이섀도 — 컬러/블렌딩
✏️ 아이라이너 — 스타일 선택
🖌️ 블러셔 — 위치/색상
🌟 하이라이터 — 광택 추가

[룩 프리셋]
- 데일리 내추럴
- 오피스 룩
- 데이트 글램
- 파티 룩
- K-뷰티 스타일
```

### 3. 조명 시뮬레이션
```
[가상 조명]
☀️ 자연광 — 창가 느낌
🔆 스튜디오 — 전문가 조명
🌅 골든아워 — 따뜻한 역광
💡 링라이트 — 아이캐치 강조
🌙 무드등 — 분위기 있게

[조명 위치]
드래그로 조명 위치 조절
강도 슬라이더
```

### 4. 포즈 & 앵글 가이드
```
[AI 포즈 추천]
- 얼굴형 분석 → 베스트 앵글 표시
- 실시간 가이드 오버레이
- "조금만 왼쪽으로" 음성 안내

[포즈 라이브러리]
- 인스타 인기 포즈
- 프로필 사진용
- 단체 셀카용
- 커플 포즈
```

### 5. 배경 & 필터
```
[배경 변경]
- 원터치 배경 제거
- AI 배경 생성
- 블러 강도 조절
- 인기 배경 템플릿

[셀카 필터]
- 톤 보정 필터 50종+
- 분위기별 카테고리
- 커스텀 필터 저장
```

### 6. 스마트 촬영
```
[음성 셔터]
"찰칵", "스마일" 등 음성 인식

[제스처 셔터]
손바닥/V 감지 → 타이머 시작

[베스트샷 선택]
연속 촬영 → AI가 베스트 컷 추천
(눈 감음, 블러 자동 제외)
```
