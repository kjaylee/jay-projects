# 📷 프로캠 매뉴얼 (ProCam Manual)

## 📋 기본 정보
- **카테고리**: DSLR급 수동 카메라
- **플랫폼**: iOS / Android
- **타겟**: 사진 애호가, 전문가, 학습자
- **예상 개발기간**: 4-5개월
- **팀 규모**: 2-4명

---

## 📱 앱 컨셉

### 한 줄 요약
> "스마트폰을 DSLR처럼! 풀 매뉴얼 컨트롤과 RAW 촬영을 지원하는 프로 카메라"

### 핵심 차별점
1. **풀 매뉴얼** — ISO, 셔터, 조리개, WB 완전 제어
2. **RAW 촬영** — DNG/ProRAW 지원
3. **히스토그램** — 실시간 노출 분석
4. **포커스 피킹** — 수동 초점 보조

---

## ⚙️ 핵심 기능

### 1. 매뉴얼 컨트롤
```
[노출 삼각형]
📊 ISO: 25 - 12800 (기기별)
⏱️ 셔터: 1/16000 - 30초
🔘 조리개: f/1.4 - f/16 (시뮬레이션)

[화이트밸런스]
☀️ 프리셋: 일광/흐림/텅스텐/형광등
🎚️ 켈빈: 2000K - 10000K
🎯 커스텀: 그레이카드 측정

[포커스]
🔄 AF/MF 전환
📍 탭 투 포커스
🔍 포커스 피킹 (Red/Yellow/Blue)
🎯 피킹 강도 조절
```

### 2. 노출 도구
```
[히스토그램]
- RGB / 루미넌스 선택
- 실시간 업데이트
- 클리핑 경고 (하이라이트/섀도우)

[제브라 패턴]
- 과노출 영역 표시
- 임계값 조절 (70-100%)

[스팟 측광]
- 탭한 지점 노출 측정
- 존 시스템 표시

[노출 보정]
-3EV ←――●――→ +3EV
```

### 3. RAW & 포맷
```
[저장 포맷]
📸 JPEG (고/중/저 품질)
🎞️ HEIF (고효율)
📄 DNG (RAW)
⭐ ProRAW (iOS)
📦 JPEG + RAW (동시)

[해상도]
- 최대 해상도
- 3:2 / 4:3 / 16:9 / 1:1
- 커스텀 크롭
```

### 4. 촬영 모드
```
[기본 모드]
📷 싱글 — 단일 촬영
🔄 연속 — 고속 연사
⏰ 타이머 — 2초/5초/10초/커스텀
📡 리모트 — Apple Watch/블루투스

[특수 모드]
🌙 장노출 — 최대 30초
⏳ 타임랩스 — 인터벌 설정
🎬 슬로모션 — 120/240fps
📐 파노라마 — 자동 스티칭
📚 브라케팅 — 노출 3/5/7장
```

### 5. 그리드 & 가이드
```
[그리드]
- 3x3 (삼분할)
- 4x4 
- 황금비율
- 대각선

[수평계]
- 틸트 표시
- 진동 피드백

[세이프 존]
- 16:9 / 4:3 / 1:1 오버레이
- 인쇄 영역 표시
```

### 6. 메타데이터 & 관리
```
[EXIF 표시]
- 촬영 설정 전체
- GPS 위치
- 렌즈 정보

[태깅]
- 커스텀 키워드
- 별점
- 컬러 라벨

[내보내기]
- AirDrop
```
