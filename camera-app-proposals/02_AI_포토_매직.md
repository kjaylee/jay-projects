# ✨ AI 포토 매직 (AI Photo Magic)

## 📋 기본 정보
- **카테고리**: AI 원터치 사진 편집
- **플랫폼**: iOS / Android
- **타겟**: 전연령, 빠른 편집 원하는 일반 사용자
- **예상 개발기간**: 4-5개월
- **팀 규모**: 2-4명

---

## 📱 앱 컨셉

### 한 줄 요약
> "터치 한 번으로 프로 사진가 수준! AI가 알아서 최적의 편집을 완성"

### 핵심 차별점
1. **원터치 AI 보정** — 복잡한 슬라이더 없음
2. **장면 인식** — 음식/인물/풍경 자동 감지
3. **Before/After** — 스와이프로 비교
4. **배치 편집** — 여러 장 한번에

---

## ⚙️ 핵심 기능

### 1. AI 자동 보정
```
[원터치 매직]
사진 선택 → "매직" 버튼 → 완성!

[AI 분석 항목]
- 노출 & 콘트라스트
- 화이트밸런스
- 채도 & 생동감
- 선명도 & 디테일
- 노이즈 제거
- 피부 보정 (인물)
- 하늘 향상 (풍경)
```

### 2. 장면별 AI
```
[자동 감지 장면]
🍔 음식 — 색감 강조, 따뜻한 톤
👤 인물 — 피부 보정, 아이 강조
🌅 풍경 — 하늘 향상, HDR
🌃 야경 — 노이즈 제거, 밝기 조절
📄 문서 — 스캔 모드, 텍스트 선명
🐕 반려동물 — 털 디테일, 눈 강조

[강도 조절]
자연스럽게 ←――●――→ 극적으로
(AI 추천값 기본 적용)
```

### 3. AI 특수 기능
```
[배경 제거/교체]
- 원터치 누끼
- AI 배경 생성
- 블러 배경 (인물 모드)

[오브젝트 제거]
- 불필요한 사람/물체 터치로 제거
- AI 인페인팅으로 자연스럽게

[AI 확대]
- 저해상도 → 고해상도
- 2x / 4x 업스케일링

[오래된 사진 복원]
- 흑백 → 컬러
- 손상 부분 복구
- 노이즈/스크래치 제거
```

### 4. 배치 편집
```
[다중 선택]
최대 50장 동시 선택
→ 동일한 AI 스타일 적용
→ 일괄 내보내기

[앨범 전체 보정]
특정 앨범 선택 → AI 자동 분류 → 각 사진 최적화
```

### 5. 비교 & 히스토리
```
[Before/After]
- 스와이프로 비교
- 분할 화면 보기
- 애니메이션 토글

[편집 히스토리]
- 모든 편집 단계 저장
- 언제든 되돌리기
- 특정 단계로 이동
```
