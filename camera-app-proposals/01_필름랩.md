# 🎞️ 필름랩 (FilmLab)

## 📋 기본 정보
- **카테고리**: 빈티지 필름 카메라
- **플랫폼**: iOS / Android
- **타겟**: 20-35세, 레트로 감성 선호층
- **예상 개발기간**: 3-4개월
- **팀 규모**: 2-3명

---

## 📱 앱 컨셉

### 한 줄 요약
> "진짜 필름 카메라처럼! 현상까지 기다려야 하는 아날로그 감성 카메라 앱"

### 핵심 차별점
1. **현상 대기 시간** — 찍은 사진을 바로 못 봄 (1-24시간)
2. **필름 롤 시스템** — 24/36장 촬영 후 현상
3. **실제 필름 시뮬레이션** — Kodak, Fuji, Ilford 감성
4. **우연의 미학** — 랜덤 광선 누출, 그레인

---

## ⚙️ 핵심 기능

### 1. 필름 롤 시스템
```
[필름 선택]
🟡 Kodak Gold 200 — 따뜻한 톤, 일상용
🔵 Kodak Portra 400 — 스킨톤 최적화
🟢 Fuji Superia 400 — 선명한 색감
⚫ Ilford HP5 — 흑백 클래식
🔴 Cinestill 800T — 시네마틱 야경

[롤 구성]
- 24장 롤 (무료)
- 36장 롤 (프리미엄)
- 각 롤 고유 특성 (색감, 그레인, 컨트라스트)
```

### 2. 현상 시스템
```
[현상 대기]
- 1시간 현상 (광고 시청)
- 3시간 현상 (기본)
- 24시간 현상 (빈티지 모드)

[현상소 선택]
🏪 동네 현상소 — 3시간, 랜덤 보정
🏢 프로 랩 — 1시간, 정확한 색감
🏚️ 오래된 현상소 — 24시간, 빈티지 결함

[결함 시뮬레이션]
- 광선 누출 (Light Leak)
- 먼지/스크래치
- 색 바램
- 프레임 겹침
```

### 3. 카메라 UI
```
[빈티지 카메라 인터페이스]
- 아날로그 다이얼 UI
- 필름 카운터 표시
- 플래시 on/off
- 셀프타이머 (물리적 사운드)

[촬영 경험]
- 셔터 사운드 (기계식)
- 필름 감기 애니메이션
- 뷰파인더 프레임
```

### 4. 갤러리 & 앨범
```
[필름 롤 앨범]
- 날짜별 롤 정리
- 현상 전/후 상태 표시
- 필름 스트립 뷰

[공유]
- 필름 스트립 이미지로 내보내기
- 인스타그램 스토리 최적화
- 프린트 주문 연동
```

## 🎨 디자인 방향

### UI/UX
- **스큐어모피즘** — 실제 카메라 느낌
- **따뜻한 색감** — 빈티지 무드
- **촉각 피드백** — 햅틱 + 사운드

- 카메라 스킨: 10종
- 현상소 테마: 5종
- 사운드 이펙트: 20종
